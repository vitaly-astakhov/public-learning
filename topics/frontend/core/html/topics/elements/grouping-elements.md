–ê—Ç—Ä–∏–±—É—Ç [`start`](https://html.spec.whatwg.org/multipage/grouping-content.html#attr-ol-start), –µ—Å–ª–∏ –ø—Ä–∏—Å—É—Ç—Å—Ç–≤—É–µ—Ç, –æ–ø—Ä–µ–¥–µ–ª—è–µ—Ç –Ω–∞—á–∞–ª—å–Ω–æ–µ —á–∏—Å–ª–æ —Å –∫–æ—Ç–æ—Ä–æ–≥–æ –Ω–∞—á–∏–Ω–∞–µ—Ç—Å—è –ø–æ—Ä—è–¥–æ–∫ –¥–ª—è —ç–ª–µ–º–µ–Ω—Ç–æ–≤, –∫–æ—Ç–æ—Ä—ã–µ –Ω–∞—Ö–æ–¥—è—Ç—Å—è –≤–Ω—É—Ç—Ä–∏ `<ol>`.

–ê—Ç—Ä–∏–±—É—Ç [`type`](https://html.spec.whatwg.org/multipage/grouping-content.html#attr-ol-type), –µ—Å–ª–∏ –ø—Ä–∏—Å—É—Ç—Å—Ç–≤—É–µ—Ç –∏ —Å–æ–æ—Ç–≤–µ—Ç—Å—Ç–≤—É–µ—Ç –æ–¥–Ω–æ–º—É –∏–∑ –∫–ª—é—á–µ–≤—ã—Ö —Å–ª–æ–≤ [–ø—Ä–µ–¥—Å—Ç–∞–≤–ª–µ–Ω–Ω—ã—Ö –≤ —Ç–∞–±–ª–∏—Ü–µ](https://html.spec.whatwg.org/multipage/grouping-content.html#attr-ol-type), –æ–ø—Ä–µ–¥–µ–ª—è–µ—Ç —Ç–∏–ø–∞ –º–∞—Ä–∫–µ—Ä–∞, –∫–æ—Ç–æ—Ä—ã–π –ø—Ä–∏–º–µ–Ω—è–µ—Ç—Å—è –¥–ª—è –º–∞—Ä–∫–∏—Ä–æ–≤–∞–Ω–∏—è —ç–ª–µ–º–µ–Ω—Ç–æ–≤, –∫–æ—Ç–æ—Ä—ã–µ –Ω–∞—Ö–æ–¥—è—Ç—Å—è –≤–Ω—É—Ç—Ä–∏ `<ol>`.
