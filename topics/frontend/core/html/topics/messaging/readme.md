## [Cross-document messaging](https://html.spec.whatwg.org/multipage/web-messaging.html#web-messaging)

–ò—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å —Å–∏—Å—Ç–µ–º—É –æ–±–º–µ–Ω–∞ —Å–æ–æ–±—â–µ–Ω–∏—è–º–∏ (*messaging system*) –º–æ–∂–Ω–æ —á–µ—Ä–µ–∑ –º–µ—Ç–æ–¥ `[window.postMessage()](https://html.spec.whatwg.org/multipage/web-messaging.html#dom-window-postmessage-options)`, –∫–æ—Ç–æ—Ä—ã–π –æ—Ç–ø—Ä–∞–≤–ª—è–µ—Ç —Å–æ–æ–±—â–µ–Ω–∏–µ –≤ —É–∫–∞–∑–∞–Ω–Ω–æ–µ –æ–∫–Ω–æ. –°–æ–æ–±—â–µ–Ω–∏—è –º–æ–≥—É—Ç –±—ã—Ç—å —Å—Ç—Ä—É–∫—Ç—É—Ä–∏—Ä–æ–≤–∞–Ω–Ω—ã–º–∏ –æ–±—ä–µ–∫—Ç–∞–º–∏ (*structured objects*), –Ω–∞–ø—Ä–∏–º–µ—Ä –≤–ª–æ–∂–µ–Ω–Ω—ã–º–∏ –æ–±—ä–µ–∫—Ç–∞–º–∏ –∏ –º–∞—Å—Å–∏–≤–∞–º–∏, –º–æ–≥—É—Ç —Å–æ–¥–µ—Ä–∂–∞—Ç—å –∑–Ω–∞—á–µ–Ω–∏—è JavaScript (—Å—Ç—Ä–æ–∫–∏, —á–∏—Å–ª–∞, –æ–±—ä–µ–∫—Ç—ã –¥–∞—Ç—ã (*`[Date](https://tc39.es/ecma262/#sec-date-objects)`¬†objects*) –∏ —Ç.–¥.) –∏ –º–æ–≥—É—Ç —Å–æ–¥–µ—Ä–∂–∞—Ç—å –æ–ø—Ä–µ–¥–µ–ª–µ–Ω–Ω—ã–µ –æ–±—ä–µ–∫—Ç—ã –¥–∞–Ω–Ω—ã—Ö.

### Cross-document messaging security

–ê–≤—Ç–æ—Ä–∞–º —Å–ª–µ–¥—É–µ—Ç –ø—Ä–æ–≤–µ—Ä–∏—Ç—å –∞—Ç—Ä–∏–±—É—Ç `origin`, —á—Ç–æ–±—ã —É–±–µ–¥–∏—Ç—å—Å—è, —á—Ç–æ —Å–æ–æ–±—â–µ–Ω–∏—è –ø—Ä–∏–Ω–∏–º–∞—é—Ç—Å—è —Ç–æ–ª—å–∫–æ —Å —Ç–µ—Ö –¥–æ–º–µ–Ω–æ–≤, —Å –∫–æ—Ç–æ—Ä—ã—Ö –æ–Ω–∏ –æ–∂–∏–¥–∞—é—Ç –ø–æ–ª—É—á–∞—Ç—å —Å–æ–æ–±—â–µ–Ω–∏—è. –í –ø—Ä–æ—Ç–∏–≤–Ω–æ–º —Å–ª—É—á–∞–µ –æ—à–∏–±–∫–∏ –≤ –∞–≤—Ç–æ—Ä—Å–∫–æ–º –∫–æ–¥–µ –æ–±—Ä–∞–±–æ—Ç–∫–∏ —Å–æ–æ–±—â–µ–Ω–∏–π –º–æ–≥—É—Ç –±—ã—Ç—å –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω—ã –≤—Ä–∞–∂–¥–µ–±–Ω—ã–º–∏ (*hostile*) —Å–∞–π—Ç–∞–º–∏.

–ö—Ä–æ–º–µ —Ç–æ–≥–æ, –¥–∞–∂–µ –ø–æ—Å–ª–µ –ø—Ä–æ–≤–µ—Ä–∫–∏ –∞—Ç—Ä–∏–±—É—Ç–∞ `origin` –∞–≤—Ç–æ—Ä—ã –¥–æ–ª–∂–Ω—ã —Ç–∞–∫–∂–µ —É–±–µ–¥–∏—Ç—å—Å—è, —á—Ç–æ –¥–∞–Ω–Ω—ã–µ, –æ –∫–æ—Ç–æ—Ä—ã—Ö –∏–¥–µ—Ç —Ä–µ—á—å, —Å–æ–æ—Ç–≤–µ—Ç—Å—Ç–≤—É—é—Ç –æ–∂–∏–¥–∞–µ–º–æ–º—É —Ñ–æ—Ä–º–∞—Ç—É. –í –ø—Ä–æ—Ç–∏–≤–Ω–æ–º —Å–ª—É—á–∞–µ, –µ—Å–ª–∏ –∏—Å—Ç–æ—á–Ω–∏–∫ —Å–æ–±—ã—Ç–∏—è –±—ã–ª –∞—Ç–∞–∫–æ–≤–∞–Ω —Å –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏–µ–º –æ—à–∏–±–∫–∏ –º–µ–∂—Å–∞–π—Ç–æ–≤–æ–≥–æ —Å–∫—Ä–∏–ø—Ç–∏–Ω–≥–∞, –¥–∞–ª—å–Ω–µ–π—à–∞—è –Ω–µ–∫–æ–Ω—Ç—Ä–æ–ª–∏—Ä—É–µ–º–∞—è –æ–±—Ä–∞–±–æ—Ç–∫–∞ –∏–Ω—Ñ–æ—Ä–º–∞—Ü–∏–∏, –æ—Ç–ø—Ä–∞–≤–ª–µ–Ω–Ω–æ–π —Å –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏–µ–º –º–µ—Ç–æ–¥–∞ `postMessage()`, –º–æ–∂–µ—Ç –ø—Ä–∏–≤–µ—Å—Ç–∏ –∫ —Ä–∞—Å–ø—Ä–æ—Å—Ç—Ä–∞–Ω–µ–Ω–∏—é –∞—Ç–∞–∫–∏ –Ω–∞ –ø–æ–ª—É—á–∞—Ç–µ–ª—è.

–ê–≤—Ç–æ—Ä–∞–º –Ω–µ —Å–ª–µ–¥—É–µ—Ç –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å –∫–ª—é—á–µ–≤–æ–µ —Å–ª–æ–≤–æ —Å –ø–æ–¥—Å—Ç–∞–Ω–æ–≤–æ—á–Ω—ã–º (*wildcard*) –∑–Ω–∞–∫–æ–º (*) –≤ –∞—Ä–≥—É–º–µ–Ω—Ç–µ [`targetOrigin`](https://html.spec.whatwg.org/multipage/web-messaging.html#dom-window-postmessage) –≤ —Å–æ–æ–±—â–µ–Ω–∏—è—Ö, —Å–æ–¥–µ—Ä–∂–∞—â–∏—Ö –∫–∞–∫—É—é-–ª–∏–±–æ –∫–æ–Ω—Ñ–∏–¥–µ–Ω—Ü–∏–∞–ª—å–Ω—É—é –∏–Ω—Ñ–æ—Ä–º–∞—Ü–∏—é, –ø–æ—Å–∫–æ–ª—å–∫—É –≤ –ø—Ä–æ—Ç–∏–≤–Ω–æ–º —Å–ª—É—á–∞–µ –Ω–µ–≤–æ–∑–º–æ–∂–Ω–æ –≥–∞—Ä–∞–Ω—Ç–∏—Ä–æ–≤–∞—Ç—å, —á—Ç–æ —Å–æ–æ–±—â–µ–Ω–∏–µ –±—É–¥–µ—Ç –¥–æ—Å—Ç–∞–≤–ª–µ–Ω–æ —Ç–æ–ª—å–∫–æ —Ç–æ–º—É –ø–æ–ª—É—á–∞—Ç–µ–ª—é, –∫–æ—Ç–æ—Ä–æ–º—É –æ–Ω–æ –ø—Ä–µ–¥–Ω–∞–∑–Ω–∞—á–∞–ª–æ—Å—å.

–ê–≤—Ç–æ—Ä–∞–º, –ø—Ä–∏–Ω–∏–º–∞—é—â–∏–º —Å–æ–æ–±—â–µ–Ω–∏—è –æ—Ç –ª—é–±–æ–≥–æ –∏—Å—Ç–æ—á–Ω–∏–∫–∞, —Ä–µ–∫–æ–º–µ–Ω–¥—É–µ—Ç—Å—è —É—á–∏—Ç—ã–≤–∞—Ç—å —Ä–∏—Å–∫–∏, —Å–≤—è–∑–∞–Ω–Ω—ã–µ —Å –∞—Ç–∞–∫–æ–π —Ç–∏–ø–∞ "–æ—Ç–∫–∞–∑ –≤ –æ–±—Å–ª—É–∂–∏–≤–∞–Ω–∏–∏" ("*denial-of-service*"). –ó–ª–æ—É–º—ã—à–ª–µ–Ω–Ω–∏–∫ –º–æ–∂–µ—Ç –æ—Ç–ø—Ä–∞–≤–∏—Ç—å –±–æ–ª—å—à–æ–µ –∫–æ–ª–∏—á–µ—Å—Ç–≤–æ —Å–æ–æ–±—â–µ–Ω–∏–π; –µ—Å–ª–∏ –ø—Ä–∏–Ω–∏–º–∞—é—â–∞—è —Å—Ç—Ä–∞–Ω–∏—Ü–∞ –≤—ã–ø–æ–ª–Ω—è–µ—Ç –¥–æ—Ä–æ–≥–æ—Å—Ç–æ—è—â–∏–µ –≤—ã—á–∏—Å–ª–µ–Ω–∏—è –∏–ª–∏ –≤—ã–∑—ã–≤–∞–µ—Ç –æ—Ç–ø—Ä–∞–≤–∫—É —Å–µ—Ç–µ–≤–æ–≥–æ —Ç—Ä–∞—Ñ–∏–∫–∞ –¥–ª—è –∫–∞–∂–¥–æ–≥–æ —Ç–∞–∫–æ–≥–æ —Å–æ–æ–±—â–µ–Ω–∏—è, —Å–æ–æ–±—â–µ–Ω–∏–µ –∑–ª–æ—É–º—ã—à–ª–µ–Ω–Ω–∏–∫–∞ –º–æ–∂–µ—Ç –±—ã—Ç—å –ø—Ä–µ–æ–±—Ä–∞–∑–æ–≤–∞–Ω–æ –≤ –∞—Ç–∞–∫—É —Ç–∏–ø–∞ "–æ—Ç–∫–∞–∑ –≤ –æ–±—Å–ª—É–∂–∏–≤–∞–Ω–∏–∏". –ê–≤—Ç–æ—Ä–∞–º —Ä–µ–∫–æ–º–µ–Ω–¥—É–µ—Ç—Å—è –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å –æ–≥—Ä–∞–Ω–∏—á–µ–Ω–∏–µ —Å–∫–æ—Ä–æ—Å—Ç–∏ (–ø—Ä–∏–Ω–∏–º–∞—Ç—å —Ç–æ–ª—å–∫–æ –æ–ø—Ä–µ–¥–µ–ª–µ–Ω–Ω–æ–µ –∫–æ–ª–∏—á–µ—Å—Ç–≤–æ —Å–æ–æ–±—â–µ–Ω–∏–π –≤ –º–∏–Ω—É—Ç—É), —á—Ç–æ–±—ã —Å–¥–µ–ª–∞—Ç—å —Ç–∞–∫–∏–µ –∞—Ç–∞–∫–∏ –Ω–µ–ø—Ä–∞–∫—Ç–∏—á–Ω—ã–º–∏.

## [Channel messaging](https://html.spec.whatwg.org/multipage/web-messaging.html#channel-messaging)
