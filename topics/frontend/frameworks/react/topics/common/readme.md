## State

**State** —è–≤–ª—è–µ—Ç—Å—è –≤–∏–¥–æ–º –ø–∞–º—è—Ç–∏, –∫–æ—Ç–æ—Ä—ã–π —Ö—Ä–∞–Ω–∏—Ç—Å—è –∏ —É–ø—Ä–∞–≤–ª—è–µ—Ç—Å—è –∫–æ–º–ø–æ–Ω–µ–Ω—Ç–æ–º. –û–Ω –∏—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è –¥–ª—è –æ—Ç—Å–ª–µ–∂–∏–≤–∞–Ω–∏—è –∏–Ω—Ñ–æ—Ä–º–∞—Ü–∏–∏, –∫–æ—Ç–æ—Ä–∞—è –º–æ–∂–µ—Ç –∏–∑–º–µ–Ω—è—Ç—å—Å—è —Å–æ –≤—Ä–µ–º–µ–Ω–µ–º.

State —Ñ–∞–∫—Ç–∏—á–µ—Å–∫–∏ ‚Äú–∂–∏–≤–µ—Ç‚Äù –≤ —Å–∞–º–æ–º React –≤–Ω–µ –≤–∞—à–µ–π —Ñ—É–Ω–∫—Ü–∏–∏ –∫–æ–º–ø–æ–Ω–µ–Ω—Ç–∞. –ö–æ–≥–¥–∞ React –≤—ã–∑—ã–≤–∞–µ—Ç –∫–æ–º–ø–æ–Ω–µ–Ω—Ç, –æ–Ω –≤—ã–¥–∞–µ—Ç –≤–∞–º —Å–Ω–∏–º–æ–∫ (*snapshot*) —Å–æ—Å—Ç–æ—è–Ω–∏—è –¥–ª—è —ç—Ç–æ–≥–æ –∫–æ–Ω–∫—Ä–µ—Ç–Ω–æ–≥–æ —Ä–µ–Ω–¥–µ—Ä–∞. –ö–æ–º–ø–æ–Ω–µ–Ω—Ç –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç —Å–Ω–∏–º–æ–∫ (*snapshot*) –ø–æ–ª—å–∑–æ–≤–∞—Ç–µ–ª—å—Å–∫–æ–≥–æ –∏–Ω—Ç–µ—Ä—Ñ–µ–π—Å–∞ —Å–æ —Å–≤–µ–∂–∏–º –Ω–∞–±–æ—Ä–æ–º props –∏ –æ–±—Ä–∞–±–æ—Ç—á–∏–∫–æ–≤ —Å–æ–±—ã—Ç–∏–π (*event handlers*) –≤ —Å–≤–æ–µ–º JSX, –≤—Å–µ –æ–Ω–∏ —Ä–∞—Å—Å—á–∏—Ç–∞–Ω—ã —Å –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏–µ–º –∑–Ω–∞—á–µ–Ω–∏–π —Å–æ—Å—Ç–æ—è–Ω–∏—è –∏–∑ —ç—Ç–æ–≥–æ —Ä–µ–Ω–¥–µ—Ä–∞!

React —Å–æ—Ö—Ä–∞–Ω—è–µ—Ç —Å–æ—Å—Ç–æ—è–Ω–∏–µ –∫–æ–º–ø–æ–Ω–µ–Ω—Ç–∞ –¥–æ —Ç–µ—Ö –ø–æ—Ä, –ø–æ–∫–∞ –æ–Ω –æ—Ç–æ–±—Ä–∞–∂–∞–µ—Ç—Å—è –≤ —Å–≤–æ–µ–º –ø–æ–ª–æ–∂–µ–Ω–∏–∏ –≤ [–¥–µ—Ä–µ–≤–µ –ø–æ–ª—å–∑–æ–≤–∞—Ç–µ–ª—å—Å–∫–æ–≥–æ –∏–Ω—Ç–µ—Ä—Ñ–µ–π—Å–∞](https://react.dev/learn/understanding-your-ui-as-a-tree#the-render-tree), —Ç–æ –µ—Å—Ç—å, –µ—Å–ª–∏ –∫–æ–º–ø–æ–Ω–µ–Ω—Ç –ø–æ–º–µ–Ω—è–µ—Ç –ø–æ–ª–æ–∂–µ–Ω–∏–µ, –∏–ª–∏ –∏–∑–º–µ–Ω–∏—Ç—Å—è –µ–≥–æ —Ä–æ–¥–∏—Ç–µ–ª—å—Å–∫–∏–π —ç–ª–µ–º–µ–Ω—Ç/–∫–æ–º–ø–æ–Ω–µ–Ω—Ç, –∏–ª–∏ –µ–≥–æ –∑–∞–º–µ–Ω–∏—Ç –¥—Ä—É–≥–æ–π —ç–ª–µ–º–µ–Ω—Ç, —Ç–æ —Å–æ—Å—Ç–æ—è–Ω–∏–µ –∫–æ–º–ø–æ–Ω–µ–Ω—Ç–∞ –Ω–µ —Å–æ—Ö—Ä–∞–Ω–∏—Ç—Å—è. –ü–æ–¥—Ä–æ–±–Ω–µ–µ –º–æ–∂–Ω–æ —É–∑–Ω–∞—Ç—å —Ç—É—Ç: <https://react.dev/learn/preserving-and-resetting-state>

### State batching

React —Å–æ—Ö—Ä–∞–Ω—è–µ—Ç –∑–Ω–∞—á–µ–Ω–∏—è —Å–æ—Å—Ç–æ—è–Ω–∏—è ‚Äú—Ñ–∏–∫—Å–∏—Ä–æ–≤–∞–Ω–Ω—ã–º–∏‚Äù –≤ –æ–±—Ä–∞–±–æ—Ç—á–∏–∫–∞—Ö —Å–æ–±—ã—Ç–∏–π –æ–¥–Ω–æ–≥–æ —Ä–µ–Ω–¥–µ—Ä–∞. –≠—Ç–æ –æ–∑–Ω–∞—á–∞–µ—Ç, —á—Ç–æ –≤–Ω—É—Ç—Ä–∏ –æ–¥–Ω–æ–≥–æ —Ä–µ–Ω–¥–µ—Ä–∞ (–æ–¥–Ω–æ–π –∏—Ç–µ—Ä–∞—Ü–∏–∏ —Ü–∏–∫–ª–∞ –∂–∏–∑–Ω–∏ –∫–æ–º–ø–æ–Ω–µ–Ω—Ç–∞) –∑–Ω–∞—á–µ–Ω–∏—è —Å–æ—Å—Ç–æ—è–Ω–∏—è (*state*) –Ω–µ –∏–∑–º–µ–Ω—è—é—Ç—Å—è, –¥–∞–∂–µ –µ—Å–ª–∏ –≤—ã–∑–≤–∞—Ç—å `setState` –Ω–µ—Å–∫–æ–ª—å–∫–æ —Ä–∞–∑ –≤ –æ–±—Ä–∞–±–æ—Ç—á–∏–∫–µ —Å–æ–±—ã—Ç–∏—è (*event handler*).

React –æ–∂–∏–¥–∞–µ—Ç, –ø–æ–∫–∞ –Ω–µ –±—É–¥–µ—Ç –≤—ã–ø–æ–ª–Ω–µ–Ω ***–≤–µ—Å—å*** –∫–æ–¥ –≤ –æ–±—Ä–∞–±–æ—Ç—á–∏–∫–∞—Ö —Å–æ–±—ã—Ç–∏–π, –ø—Ä–µ–∂–¥–µ —á–µ–º –æ–±–Ω–æ–≤–ª—è—Ç—å —Å–æ—Å—Ç–æ—è–Ω–∏–µ –∫–æ–º–ø–æ–Ω–µ–Ω—Ç–∞, —ç—Ç–æ –Ω–∞–∑—ã–≤–∞–µ—Ç—Å—è [**batching**](https://react.dev/learn/queueing-a-series-of-state-updates#react-batches-state-updates).

–ü–µ—Ä–µ–¥–∞—á–∞ –∑–Ω–∞—á–µ–Ω–∏–π –≤ —Ñ—É–Ω–∫—Ü–∏—é set (`setState`) —Å–æ–∑–¥–∞–µ—Ç –æ—á–µ—Ä–µ–¥—å –æ–±–Ω–æ–≤–ª–µ–Ω–∏–π (*update queue*). –ó–Ω–∞—á–µ–Ω–∏–µ –º–æ–∂–µ—Ç –ª–∏–±–æ –∑–∞–º–µ–Ω—è—Ç—å (**replace**) —Ç–µ–∫—É—â–µ–µ –∑–Ω–∞—á–µ–Ω–∏–µ —Å–æ—Å—Ç–æ—è–Ω–∏—è (*state*), –∫–æ—Ç–æ—Ä–æ–µ –Ω–∞—Ö–æ–¥–∏—Ç—Å—è –≤ –æ—á–µ—Ä–µ–¥–∏ (–Ω–∞–ø—Ä–∏–º–µ—Ä, `setState(replacingState)`), –ª–∏–±–æ –¥–æ–±–∞–≤–ª—è—Ç—å —Å–ª–µ–¥—É—é—â–µ–µ —Å–æ—Å—Ç–æ—è–Ω–∏–µ, –æ—Å–Ω–æ–≤–∞–Ω–Ω–æ–µ –Ω–∞ –ø—Ä–µ–¥—ã–¥—É—â–µ–º —Å–æ—Å—Ç–æ—è–Ω–∏–∏ –≤ –æ—á–µ—Ä–µ–¥–∏, —á–µ—Ä–µ–∑ —Ñ—É–Ω–∫—Ü–∏—é –æ–±–Ω–æ–≤–ª–µ–Ω–∏—è (**updater function**) (–Ω–∞–ø—Ä–∏–º–µ—Ä, `setState(prevState => prevState + newState)`).

–ü–æ–¥—Ä–æ–±–Ω–µ–µ –ø—Ä–æ **state batching** –º–æ–∂–Ω–æ —É–∑–Ω–∞—Ç—å —Ç—É—Ç: [Queueing a Series of State Updates](https://react.dev/learn/queueing-a-series-of-state-updates)
