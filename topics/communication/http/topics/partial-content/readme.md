## Overview

**Range requests (–¥–∏–∞–ø–∞–∑–æ–Ω–Ω—ã–µ –∑–∞–ø—Ä–æ—Å—ã)** ‚Äî —ç—Ç–æ –û–ü–¶–ò–û–ù–ê–õ–¨–ù–ê–Ø —Ñ—É–Ω–∫—Ü–∏—è HTTP, —Ä–∞–∑—Ä–∞–±–æ—Ç–∞–Ω–Ω–∞—è —Ç–∞–∫–∏–º –æ–±—Ä–∞–∑–æ–º, —á—Ç–æ–±—ã –ø–æ–ª—É—á–∞—Ç–µ–ª–∏, –Ω–µ —Ä–µ–∞–ª–∏–∑—É—é—â–∏–µ —ç—Ç—É —Ñ—É–Ω–∫—Ü–∏—é (–∏–ª–∏ –Ω–µ –ø–æ–¥–¥–µ—Ä–∂–∏–≤–∞—é—â–∏–µ –µ–µ –¥–ª—è —Ü–µ–ª–µ–≤–æ–≥–æ —Ä–µ—Å—É—Ä—Å–∞), –º–æ–≥–ª–∏ –æ—Ç–≤–µ—á–∞—Ç—å, –∫–∞–∫ –µ—Å–ª–∏ –±—ã —ç—Ç–æ –±—ã–ª –æ–±—ã—á–Ω—ã–π –∑–∞–ø—Ä–æ—Å GET, –Ω–µ –≤–ª–∏—è—è –Ω–∞ —Å–æ–≤–º–µ—Å—Ç–∏–º–æ—Å—Ç—å. –ß–∞—Å—Ç–∏—á–Ω—ã–µ –æ—Ç–≤–µ—Ç—ã –æ–±–æ–∑–Ω–∞—á–∞—é—Ç—Å—è –æ—Ç–¥–µ–ª—å–Ω—ã–º –∫–æ–¥–æ–º —Å–æ—Å—Ç–æ—è–Ω–∏—è ([206 Partial Content](https://www.rfc-editor.org/rfc/rfc9110.html#name-206-partial-content)), —á—Ç–æ–±—ã –∫—ç—à–∏, –∫–æ—Ç–æ—Ä—ã–µ –º–æ–≥—É—Ç –Ω–µ —Ä–µ–∞–ª–∏–∑–æ–≤–∞—Ç—å —ç—Ç—É —Ñ—É–Ω–∫—Ü–∏—é, –Ω–µ –ø—Ä–∏–Ω—è–ª–∏ –∏—Ö –∑–∞ –ø–æ–ª–Ω—ã–µ –æ—Ç–≤–µ—Ç—ã.

–ü–æ–ª–Ω—ã–π —Å–ø–∏—Å–æ–∫ –∑–∞—Ä–µ–≥–∏—Å—Ç—Ä–∏—Ä–æ–≤–∞–Ω–Ω—ã—Ö –¥–∏–∞–ø–∞–∑–æ–Ω–Ω—ã—Ö –µ–¥–∏–Ω–∏—Ü: <https://www.iana.org/assignments/http-parameters/http-parameters.xhtml#range-units>
