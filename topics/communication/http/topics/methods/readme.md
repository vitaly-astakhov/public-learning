## Overview

**Request method** - —á–∞—Å—Ç—å request —Å–µ–º–∞–Ω—Ç–∏–∫–∏ –∏ —É–∫–∞–∑—ã–≤–∞–µ—Ç –Ω–∞ —Ü–µ–ª—å/–¥–µ–π—Å—Ç–≤–∏–µ, —Å –∫–æ—Ç–æ—Ä–æ–π –∫–ª–∏–µ–Ω—Ç —Å–¥–µ–ª–∞–ª —ç—Ç–æ—Ç –∑–∞–ø—Ä–æ—Å, –∏ —á—Ç–æ –æ–∂–∏–¥–∞–µ—Ç—Å—è –∫–ª–∏–µ–Ω—Ç–æ–º –≤ –∫–∞—á–µ—Å—Ç–≤–µ —É—Å–ø–µ—à–Ω–æ–≥–æ —Ä–µ–∑—É–ª—å—Ç–∞—Ç–∞.

Request method –≤—ã–∑—ã–≤–∞–µ—Ç –¥–µ–π—Å—Ç–≤–∏–µ, –∫–æ—Ç–æ—Ä–æ–µ –¥–æ–ª–∂–Ω–æ –±—ã—Ç—å –ø—Ä–∏–º–µ–Ω–µ–Ω–æ –∫ —Ü–µ–ª–µ–≤–æ–º—É —Ä–µ—Å—É—Ä—Å—É.

### Method safeness

–ú–µ—Ç–æ–¥—ã —Å—á–∏—Ç–∞—é—Ç—Å—è **–±–µ–∑–æ–ø–∞—Å–Ω—ã–º–∏ (*safe*)**, –µ—Å–ª–∏ –∏—Ö —Å–µ–º–∞–Ω—Ç–∏–∫–∞ –æ–∑–Ω–∞—á–∞–µ—Ç —Ç–æ–ª—å–∫–æ –¥–æ—Å—Ç—É–ø –∫ —á—Ç–µ–Ω–∏—é —Ü–µ–ª–µ–≤–æ–≥–æ —Ä–µ—Å—É—Ä—Å–∞. –ë–µ–∑–æ–ø–∞—Å–Ω—ã–º–∏ –º–µ—Ç–æ–¥–∞–º–∏ —è–≤–ª—è—é—Ç—Å—è: `GET`, `HEAD`, `OPTIONS`, and `TRACE`. [^1]

–¶–µ–ª—å –ø—Ä–æ–≤–µ–¥–µ–Ω–∏—è —Ä–∞–∑–ª–∏—á–∏—è –º–µ–∂–¥—É –±–µ–∑–æ–ø–∞—Å–Ω—ã–º–∏ –∏ –Ω–µ–±–µ–∑–æ–ø–∞—Å–Ω—ã–º–∏ –º–µ—Ç–æ–¥–∞–º–∏ —Å–æ—Å—Ç–æ–∏—Ç –≤ —Ç–æ–º, —á—Ç–æ–±—ã –ø–æ–∑–≤–æ–ª–∏—Ç—å –∞–≤—Ç–æ–º–∞—Ç–∏–∑–∏—Ä–æ–≤–∞–Ω–Ω—ã–º –ø—Ä–æ—Ü–µ—Å—Å–∞–º –ø–æ–∏—Å–∫–∞ (spiders, web-crawlers) –∏ –æ–ø—Ç–∏–º–∏–∑–∞—Ü–∏–∏ –ø—Ä–æ–∏–∑–≤–æ–¥–∏—Ç–µ–ª—å–Ω–æ—Å—Ç–∏ –∫—ç—à–∞ (–ø—Ä–µ–¥–≤–∞—Ä–∏—Ç–µ–ª—å–Ω–∞—è –≤—ã–±–æ—Ä–∫–∞ - pre-fetching) —Ä–∞–±–æ—Ç–∞—Ç—å, –Ω–µ –æ–ø–∞—Å–∞—è—Å—å –ø—Ä–∏—á–∏–Ω–µ–Ω–∏—è –≤—Ä–µ–¥–∞.

### Idempotency

[–ò–¥–µ–º–ø–æ—Ç–µ–Ω—Ç–Ω—ã–π –º–µ—Ç–æ–¥](https://www.rfc-editor.org/rfc/rfc9110#name-idempotent-methods) - —ç—Ç–æ –º–µ—Ç–æ–¥, –∫–æ—Ç–æ—Ä—ã–π –º–æ–∂–µ—Ç –±—ã—Ç—å –≤—ã–∑–≤–∞–Ω –º–Ω–æ–≥–æ —Ä–∞–∑ –±–µ–∑ —Ä–∞–∑–Ω—ã—Ö —Ä–µ–∑—É–ª—å—Ç–∞—Ç–æ–≤. –ù–µ –¥–æ–ª–∂–Ω–æ –∏–º–µ—Ç—å –∑–Ω–∞—á–µ–Ω–∏—è, –±—ã–ª –ª–∏ –º–µ—Ç–æ–¥ –≤—ã–∑–≤–∞–Ω —Ç–æ–ª—å–∫–æ –æ–¥–∏–Ω —Ä–∞–∑ –∏–ª–∏ –±–æ–ª–µ–µ –¥–µ—Å—è—Ç–∏ —Ä–∞–∑. –†–µ–∑—É–ª—å—Ç–∞—Ç –≤—Å–µ–≥–¥–∞ –¥–æ–ª–∂–µ–Ω –±—ã—Ç—å –æ–¥–∏–Ω–∞–∫–æ–≤—ã–º.
–ò–¥–µ–º–ø–æ—Ç–µ–Ω—Ç–Ω—ã–º–∏ –º–µ—Ç–æ–¥–∞–º–∏ —è–≤–ª—è—é—Ç—Å—è: `PUT`,`DELETE`. –¢–∞–∫ –∂–µ –∏–¥–µ–º–ø–æ—Ç–µ–Ω—Ç–Ω—ã–º–∏ –º–µ—Ç–æ–¥–∞–º–∏ —Å—á–∏—Ç–∞—é—Ç—Å—è "–±–µ–∑–æ–ø–∞—Å–Ω—ã–µ" –º–µ—Ç–æ–¥—ã:`GET`,`HEAD`,`OPTIONS`,`TRACE`.

–ú–µ—Ç–æ–¥ `POST` –Ω–µ –∏–º–ø–æ—Ç–µ–Ω—Ç–Ω—ã–π —Ç–∞–∫ –∫–∞–∫ –æ–Ω –æ–±—ã—á–Ω–æ, –Ω–æ –Ω–µ –æ–±—è–∑–∞—Ç–µ–ª—å–Ω–æ, –∏—Å–ø–æ–ª—å–∑—É—é—Ç—Å—è –¥–ª—è —Å–æ–∑–¥–∞–Ω–∏—è –Ω–æ–≤–æ–≥–æ —Ä–µ—Å—É—Ä—Å–∞ –Ω–∞ —Å–µ—Ä–≤–µ—Ä–µ. –¢–∞–∫–∏–º –æ–±—Ä–∞–∑–æ–º, –∫–æ–≥–¥–∞ –º—ã –≤—ã–ø–æ–ª–Ω—è–µ–º –æ–¥–∏–Ω –∏ —Ç–æ—Ç –∂–µ `POST` –∑–∞–ø—Ä–æ—Å N-–∫–æ–ª–∏—á–µ—Å—Ç–≤–æ —Ä–∞–∑, —É –Ω–∞—Å –±—É–¥–µ—Ç N-–∫–æ–ª–∏—á–µ—Å—Ç–≤–æ –Ω–æ–≤—ã—Ö —Ä–µ—Å—É—Ä—Å–æ–≤ –Ω–∞ —Å–µ—Ä–≤–µ—Ä–µ.
