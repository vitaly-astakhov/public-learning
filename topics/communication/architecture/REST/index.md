**REpresentational State Transfer (REST)** - –∞—Ä—Ö–∏—Ç–µ–∫—Ç—É—Ä–Ω—ã–π —Å—Ç–∏–ª—å, –æ–ø–∏—Å—ã–≤–∞—é—â–∏–π –ø—Ä–∏–Ω—Ü–∏–ø—ã —Ä–∞–∑—Ä–∞–±–æ—Ç–∫–∏ –ø—Ä–æ–≥—Ä–∞–º–º–Ω–æ–≥–æ –æ–±–µ—Å–ø–µ—á–µ–Ω–∏—è, –ª–µ–∂–∞—â–∏–µ –≤ –æ—Å–Ω–æ–≤–µ **REST**, –∏ –æ–≥—Ä–∞–Ω–∏—á–µ–Ω–∏—è (**restrictions**) –≤–∑–∞–∏–º–æ–¥–µ–π—Å—Ç–≤–∏—è, –≤—ã–±—Ä–∞–Ω–Ω—ã–µ –¥–ª—è —Å–æ—Ö—Ä–∞–Ω–µ–Ω–∏—è —ç—Ç–∏—Ö –ø—Ä–∏–Ω—Ü–∏–ø–æ–≤.

–≠—Ç–∏ –æ–≥—Ä–∞–Ω–∏—á–µ–Ω–∏—è –≤ —Å–µ–±—è –≤–∫–ª—é—á–∞—é—Ç:

- Client-server model
- Statelessness
- Cache-ability
- Uniform interface
- Layered system
- Code-on-demand (optional)

–û–≥—Ä–∞–Ω–∏—á–µ–Ω–∏—è —Ö–æ—Ä–æ—à–æ –æ–ø–∏—Å–∞–Ω—ã –≤ —Å—Ç–∞—Ç—å–µ [Rethinking REST](https://kieranpotts.com/rethinking-rest#_the_rest_constraints).

**RESTful** - —ç—Ç–æ –≤–µ–±-—Å–µ—Ä–≤–∏—Å –∏–ª–∏ API, —Ä–µ–∞–ª–∏–∑–æ–≤–∞–Ω–Ω—ã–π —Å –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏–µ–º **HTTP** –∏ –ø—Ä–∏–Ω—Ü–∏–ø–æ–≤ **REST**.

___

–ì–ª–∞–≤–Ω–æ–µ, —á—Ç–æ –æ–ø—Ä–µ–¥–µ–ª—è–µ—Ç **REST**, —ç—Ç–æ —Ç–æ —á—Ç–æ –ø–æ–ª—å–∑–æ–≤–∞—Ç–µ–ª—å –¥–æ–ª–∂–µ–Ω –ø–æ–ª—É—á–∞—Ç—å –Ω–µ —Ä–µ—Å—É—Ä—Å (**resource**), –∞ –µ–≥–æ –ø—Ä–µ–¥—Å—Ç–∞–≤–ª–µ–Ω–∏–µ (**representation**).

–ö–æ–º–ø–æ–Ω–µ–Ω—Ç—ã **REST** –≤—ã–ø–æ–ª–Ω—è—é—Ç –¥–µ–π—Å—Ç–≤–∏—è —Å —Ä–µ—Å—É—Ä—Å–æ–º (**resource**), –∏—Å–ø–æ–ª—å–∑—É—è –ø—Ä–µ–¥—Å—Ç–∞–≤–ª–µ–Ω–∏–µ (**representation**) –¥–ª—è —Ñ–∏–∫—Å–∞—Ü–∏–∏ —Ç–µ–∫—É—â–µ–≥–æ –∏–ª–∏ –ø—Ä–µ–¥–ø–æ–ª–∞–≥–∞–µ–º–æ–≥–æ —Å–æ—Å—Ç–æ—è–Ω–∏—è —ç—Ç–æ–≥–æ —Ä–µ—Å—É—Ä—Å–∞ –∏ –ø–µ—Ä–µ–¥–∞—á–∏ —ç—Ç–æ–≥–æ –ø—Ä–µ–¥—Å—Ç–∞–≤–ª–µ–Ω–∏—è (**representation**) –º–µ–∂–¥—É –∫–æ–º–ø–æ–Ω–µ–Ω—Ç–∞–º–∏.

–ü—Ä–µ–¥—Å—Ç–∞–≤–ª–µ–Ω–∏–µ (**representation**) —Å–æ—Å—Ç–æ–∏—Ç –∏–∑ –¥–∞–Ω–Ω—ã—Ö (**data**), –º–µ—Ç–∞–¥–∞–Ω–Ω—ã—Ö (**metadata**), –æ–ø–∏—Å—ã–≤–∞—é—â–∏—Ö –¥–∞–Ω–Ω—ã–µ, –∏, –∏–Ω–æ–≥–¥–∞, –º–µ—Ç–∞–¥–∞–Ω–Ω—ã—Ö –¥–ª—è –æ–ø–∏—Å–∞–Ω–∏—è –º–µ—Ç–∞–¥–∞–Ω–Ω—ã—Ö (–æ–±—ã—á–Ω–æ —Å —Ü–µ–ª—å—é –ø—Ä–æ–≤–µ—Ä–∫–∏ —Ü–µ–ª–æ—Å—Ç–Ω–æ—Å—Ç–∏ —Å–æ–æ–±—â–µ–Ω–∏—è).

- **client** - –∫–ª–∏–µ–Ω—Ç –∏–Ω–∏—Ü–∏–∏—Ä—É–µ—Ç —Å–≤—è–∑—å, –æ—Ç–ø—Ä–∞–≤–ª—è—è –∑–∞–ø—Ä–æ—Å (**request**) - [**Connections, Clients, and Servers [HTTP Semantics]**](https://www.rfc-editor.org/rfc/rfc9110#section-3.3)
- **server** - –ø—Ä–æ—Å–ª—É—à–∏–≤–∞–µ—Ç —Å–æ–µ–¥–∏–Ω–µ–Ω–∏—è –∏ –æ—Ç–≤–µ—á–∞–µ—Ç –Ω–∞ –∑–∞–ø—Ä–æ—Å—ã (**request**), —á—Ç–æ–±—ã –ø—Ä–µ–¥–æ—Å—Ç–∞–≤–∏—Ç—å –¥–æ—Å—Ç—É–ø –∫ —Å–≤–æ–∏–º —Å–ª—É–∂–±–∞–º. - [**Connections, Clients, and Servers [HTTP Semantics]**](https://www.rfc-editor.org/rfc/rfc9110#section-3.3)
- **cache** - –º–æ–∂–µ—Ç –±—ã—Ç—å —Ä–∞—Å–ø–æ–ª–æ–∂–µ–Ω –Ω–∞ –∏–Ω—Ç–µ—Ä—Ñ–µ–π—Å–µ –∫–ª–∏–µ–Ω—Ç—Å–∫–æ–≥–æ –∏–ª–∏ —Å–µ—Ä–≤–µ—Ä–Ω–æ–≥–æ —Å–æ–µ–¥–∏–Ω–∏—Ç–µ–ª—è –¥–ª—è —Å–æ—Ö—Ä–∞–Ω–µ–Ω–∏—è –∫—ç—à–∏—Ä—É–µ–º—ã—Ö –æ—Ç–≤–µ—Ç–æ–≤ –Ω–∞ —Ç–µ–∫—É—â–∏–µ –≤–∑–∞–∏–º–æ–¥–µ–π—Å—Ç–≤–∏—è, —á—Ç–æ–±—ã –∏—Ö –º–æ–∂–Ω–æ –±—ã–ª–æ –ø–æ–≤—Ç–æ—Ä–Ω–æ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å –¥–ª—è –ø–æ—Å–ª–µ–¥—É—é—â–∏—Ö –∑–∞–ø—Ä–æ—à–µ–Ω–Ω—ã—Ö –≤–∑–∞–∏–º–æ–¥–µ–π—Å—Ç–≤–∏–π. –ö—ç—à –º–æ–∂–µ—Ç –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å—Å—è –∫–ª–∏–µ–Ω—Ç–æ–º (**client**), —á—Ç–æ–±—ã –∏–∑–±–µ–∂–∞—Ç—å –ø–æ–≤—Ç–æ—Ä–µ–Ω–∏—è —Å–µ—Ç–µ–≤–æ–≥–æ –≤–∑–∞–∏–º–æ–¥–µ–π—Å—Ç–≤–∏—è, –∏–ª–∏ —Å–µ—Ä–≤–µ—Ä–æ–º (**server**), —á—Ç–æ–±—ã –∏–∑–±–µ–∂–∞—Ç—å –ø–æ–≤—Ç–æ—Ä–µ–Ω–∏—è –ø—Ä–æ—Ü–µ—Å—Å–∞ –≥–µ–Ω–µ—Ä–∞—Ü–∏–∏ –æ—Ç–≤–µ—Ç–∞, –ø—Ä–∏—á–µ–º –æ–±–∞ —Å–ª—É—á–∞—è —Å–ª—É–∂–∞—Ç –¥–ª—è —É–º–µ–Ω—å—à–µ–Ω–∏—è –∑–∞–¥–µ—Ä–∂–∫–∏ –≤–∑–∞–∏–º–æ–¥–µ–π—Å—Ç–≤–∏—è. –ù–µ–∫–æ—Ç–æ—Ä—ã–µ —Å–æ–µ–¥–∏–Ω–∏—Ç–µ–ª–∏ –∫—ç—à–∞ —è–≤–ª—è—é—Ç—Å—è –æ–±—â–∏–º–∏ (**shared**), —á—Ç–æ –æ–∑–Ω–∞—á–∞–µ—Ç, —á—Ç–æ –∏—Ö –∫—ç—à–∏—Ä–æ–≤–∞–Ω–Ω—ã–µ –æ—Ç–≤–µ—Ç—ã –º–æ–≥—É—Ç –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å—Å—è –≤ –æ—Ç–≤–µ—Ç–µ –∫–ª–∏–µ–Ω—Ç—É, –æ—Ç–ª–∏—á–Ω–æ–º—É –æ—Ç —Ç–æ–≥–æ, –¥–ª—è –∫–æ—Ç–æ—Ä–æ–≥–æ –æ—Ç–≤–µ—Ç –±—ã–ª –ø–µ—Ä–≤–æ–Ω–∞—á–∞–ª—å–Ω–æ –ø–æ–ª—É—á–µ–Ω. - [**Caches [HTTP Semantics]**](https://www.rfc-editor.org/rfc/rfc9110#section-3.8), [**HTTP Cache üìÇ**](../../http/topics/cache.md)
- **resolver** - –ø—Ä–µ–æ–±—Ä–∞–∑—É–µ—Ç —á–∞—Å—Ç–∏—á–Ω—ã–µ –∏–ª–∏ –ø–æ–ª–Ω—ã–µ –∏–¥–µ–Ω—Ç–∏—Ñ–∏–∫–∞—Ç–æ—Ä—ã —Ä–µ—Å—É—Ä—Å–æ–≤ (**resource identifiers**) –≤ –∏–Ω—Ñ–æ—Ä–º–∞—Ü–∏—é –æ —Å–µ—Ç–µ–≤–æ–º –∞–¥—Ä–µ—Å–µ, –Ω–µ–æ–±—Ö–æ–¥–∏–º—É—é –¥–ª—è —É—Å—Ç–∞–Ω–æ–≤–ª–µ–Ω–∏—è –º–µ–∂–∫–æ–º–ø–æ–Ω–µ–Ω—Ç–Ω–æ–≥–æ —Å–æ–µ–¥–∏–Ω–µ–Ω–∏—è. –ù–∞–ø—Ä–∏–º–µ—Ä, –±–æ–ª—å—à–∏–Ω—Å—Ç–≤–æ URI –≤–∫–ª—é—á–∞—é—Ç –∏–º—è —Ö–æ—Å—Ç–∞ DNS –≤ –∫–∞—á–µ—Å—Ç–≤–µ –º–µ—Ö–∞–Ω–∏–∑–º–∞ –∏–¥–µ–Ω—Ç–∏—Ñ–∏–∫–∞—Ü–∏–∏ –æ—Ä–≥–∞–Ω–∞ —É–ø—Ä–∞–≤–ª–µ–Ω–∏—è (**authority**) –ø–æ –∏–º–µ–Ω–æ–≤–∞–Ω–∏—é —Ä–µ—Å—É—Ä—Å–∞. –ß—Ç–æ–±—ã –∏–Ω–∏—Ü–∏–∏—Ä–æ–≤–∞—Ç—å –∑–∞–ø—Ä–æ—Å, –≤–µ–±-–±—Ä–∞—É–∑–µ—Ä –∏–∑–≤–ª–µ–∫–∞–µ—Ç –∏–º—è —Ö–æ—Å—Ç–∞ –∏–∑ URI –∏ –∏—Å–ø–æ–ª—å–∑—É–µ—Ç –ø—Ä–µ–æ–±—Ä–∞–∑–æ–≤–∞—Ç–µ–ª—å (**resolver**) DNS –¥–ª—è –ø–æ–ª—É—á–µ–Ω–∏—è –∞–¥—Ä–µ—Å–∞ –ò–Ω—Ç–µ—Ä–Ω–µ—Ç-–ø—Ä–æ—Ç–æ–∫–æ–ª–∞ –¥–ª—è —ç—Ç–æ–≥–æ –æ—Ä–≥–∞–Ω–∞ —É–ø—Ä–∞–≤–ª–µ–Ω–∏—è (**authority**).
- **tunnel** - –ø–µ—Ä–µ–¥–∞–µ—Ç —Å–≤—è–∑—å —á–µ—Ä–µ–∑ –≥—Ä–∞–Ω–∏—Ü—É —Å–æ–µ–¥–∏–Ω–µ–Ω–∏—è, —Ç–∞–∫—É—é –∫–∞–∫ firewall –∏–ª–∏ —Å–µ—Ç–µ–≤–æ–π —à–ª—é–∑ (*network gateway*) –±–æ–ª–µ–µ –Ω–∏–∑–∫–æ–≥–æ —É—Ä–æ–≤–Ω—è. –ï–¥–∏–Ω—Å—Ç–≤–µ–Ω–Ω–∞—è –ø—Ä–∏—á–∏–Ω–∞, –ø–æ –∫–æ—Ç–æ—Ä–æ–π —Å–æ–µ–¥–∏–Ω–∏—Ç–µ–ª—å **tunnel** –º–æ–¥–µ–ª–∏—Ä—É–µ—Ç—Å—è –∫–∞–∫ —á–∞—Å—Ç—å **REST**, –∞ –Ω–µ –∞–±—Å—Ç—Ä–∞–≥–∏—Ä—É–µ—Ç—Å—è –∫–∞–∫ —á–∞—Å—Ç—å —Å–µ—Ç–µ–≤–æ–π –∏–Ω—Ñ—Ä–∞—Å—Ç—Ä—É–∫—Ç—É—Ä—ã, –∑–∞–∫–ª—é—á–∞–µ—Ç—Å—è –≤ —Ç–æ–º, —á—Ç–æ –Ω–µ–∫–æ—Ç–æ—Ä—ã–µ –∫–æ–º–ø–æ–Ω–µ–Ω—Ç—ã **REST** –º–æ–≥—É—Ç –¥–∏–Ω–∞–º–∏—á–µ—Å–∫–∏ –ø–µ—Ä–µ–∫–ª—é—á–∞—Ç—å—Å—è —Å –ø–æ–≤–µ–¥–µ–Ω–∏—è –∞–∫—Ç–∏–≤–Ω–æ–≥–æ –∫–æ–º–ø–æ–Ω–µ–Ω—Ç–∞ –Ω–∞ –ø–æ–≤–µ–¥–µ–Ω–∏–µ —Ç—É–Ω–Ω–µ–ª—è. - [**Intermediaries [HTTP Semantics]**](https://www.rfc-editor.org/rfc/rfc9110#section-3.7)
