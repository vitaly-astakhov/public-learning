**Slices** (—Å—Ä–µ–∑—ã) - —ç—Ç–æ –≤—Ç–æ—Ä–æ–π —É—Ä–æ–≤–µ–Ω—å –≤ –æ—Ä–≥–∞–Ω–∏–∑–∞—Ü–∏–æ–Ω–Ω–æ–π –∏–µ—Ä–∞—Ä—Ö–∏–∏ —Ñ—É–Ω–∫—Ü–∏–æ–Ω–∞–ª—å–Ω–æ-—Å–µ–≥–º–µ–Ω—Ç–∏—Ä–æ–≤–∞–Ω–Ω–æ–≥–æ –¥–∏–∑–∞–π–Ω–∞ (*Feature-Sliced Design*). –ò—Ö –æ—Å–Ω–æ–≤–Ω–∞—è —Ü–µ–ª—å - —Å–≥—Ä—É–ø–ø–∏—Ä–æ–≤–∞—Ç—å –∫–æ–¥ –ø–æ –µ–≥–æ –∑–Ω–∞—á–µ–Ω–∏—é –¥–ª—è –ø—Ä–æ–¥—É–∫—Ç–∞, –±–∏–∑–Ω–µ—Å–∞ –∏–ª–∏ –ø—Ä–æ—Å—Ç–æ –ø—Ä–∏–ª–æ–∂–µ–Ω–∏—è.

–ù–∞–∑–≤–∞–Ω–∏—è —Ñ—Ä–∞–≥–º–µ–Ω—Ç–æ–≤ –Ω–µ —Å—Ç–∞–Ω–¥–∞—Ä—Ç–∏–∑–∏—Ä–æ–≤–∞–Ω—ã, –ø–æ—Å–∫–æ–ª—å–∫—É –æ–Ω–∏ –Ω–∞–ø—Ä—è–º—É—é –æ–ø—Ä–µ–¥–µ–ª—è—é—Ç—Å—è –±–∏–∑–Ω–µ—Å-–æ–±–ª–∞—Å—Ç—å—é (*business domain*) –≤–∞—à–µ–≥–æ –ø—Ä–∏–ª–æ–∂–µ–Ω–∏—è. –ù–∞–ø—Ä–∏–º–µ—Ä, –≤ —Ñ–æ—Ç–æ–≥–∞–ª–µ—Ä–µ–µ –º–æ–≥—É—Ç –±—ã—Ç—å —Ä–∞–∑–¥–µ–ª—ã `photo`, `create-album`, `gallery-page`. –î–ª—è —Å–æ—Ü–∏–∞–ª—å–Ω–æ–π —Å–µ—Ç–∏ –º–æ–≥—É—Ç –ø–æ—Ç—Ä–µ–±–æ–≤–∞—Ç—å—Å—è —Ä–∞–∑–Ω—ã–µ —Ä–∞–∑–¥–µ–ª—ã, –Ω–∞–ø—Ä–∏–º–µ—Ä, `post`, `add-user-to-friends`, `news-feed`.

–¢–µ—Å–Ω–æ —Å–≤—è–∑–∞–Ω–Ω—ã–µ —Å—Ä–µ–∑—ã (*slices*) –º–æ–≥—É—Ç –±—ã—Ç—å —Å—Ç—Ä—É–∫—Ç—É—Ä–Ω–æ —Å–≥—Ä—É–ø–ø–∏—Ä–æ–≤–∞–Ω—ã –≤ –∫–∞—Ç–∞–ª–æ–≥–µ, –Ω–æ –æ–Ω–∏ –¥–æ–ª–∂–Ω—ã —Å–æ–æ—Ç–≤–µ—Ç—Å—Ç–≤–æ–≤–∞—Ç—å —Ç–µ–º –∂–µ –ø—Ä–∞–≤–∏–ª–∞–º –∏–∑–æ–ª—è—Ü–∏–∏, —á—Ç–æ –∏ –¥—Ä—É–≥–∏–µ —Å—Ä–µ–∑—ã (*slices*) ‚Äî –≤ —ç—Ç–æ–º –∫–∞—Ç–∞–ª–æ–≥–µ (*directory*) **–Ω–µ –¥–æ–ª–∂–Ω–æ –±—ã—Ç—å —Å–æ–≤–º–µ—Å—Ç–Ω–æ–≥–æ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏—è –∫–æ–¥–∞** (***no code sharing***).

–°–ª–æ–∏ **Shared** –∏ **App** –Ω–µ —Å–æ–¥–µ—Ä–∂–∞—Ç —Ñ—Ä–∞–≥–º–µ–Ω—Ç–æ–≤. –≠—Ç–æ —Å–≤—è–∑–∞–Ω–æ —Å —Ç–µ–º, —á—Ç–æ —Å–ª–æ–π **Shared** –≤–æ–æ–±—â–µ –Ω–µ –¥–æ–ª–∂–µ–Ω —Å–æ–¥–µ—Ä–∂–∞—Ç—å –±–∏–∑–Ω–µ—Å-–ª–æ–≥–∏–∫—É, —Å–ª–µ–¥–æ–≤–∞—Ç–µ–ª—å–Ω–æ, –Ω–µ –∏–º–µ–µ—Ç –∑–Ω–∞—á–µ–Ω–∏—è –¥–ª—è –ø—Ä–æ–¥—É–∫—Ç–∞, –∞ —Å–ª–æ–π **App** –¥–æ–ª–∂–Ω–æ —Å–æ–¥–µ—Ä–∂–∞—Ç—å —Ç–æ–ª—å–∫–æ –∫–æ–¥, –æ—Ç–Ω–æ—Å—è—â–∏–π—Å—è –∫–æ –≤—Å–µ–º—É –ø—Ä–∏–ª–æ–∂–µ–Ω–∏—é, –ø–æ—ç—Ç–æ–º—É —Ä–∞–∑–±–∏–µ–Ω–∏–µ –Ω–µ —Ç—Ä–µ–±—É–µ—Ç—Å—è.

–í–Ω—É—Ç—Ä–∏ —Å—Ä–µ–∑–∞ (*slice*) –∫–æ–¥ –º–æ–∂–µ—Ç –±—ã—Ç—å –æ—Ä–≥–∞–Ω–∏–∑–æ–≤–∞–Ω –æ—á–µ–Ω—å —Å–≤–æ–±–æ–¥–Ω–æ, –∏ —ç—Ç–æ –Ω–µ —Å–æ–∑–¥–∞–µ—Ç –Ω–∏–∫–∞–∫–∏—Ö –ø—Ä–æ–±–ª–µ–º, –µ—Å–ª–∏ —Ñ—Ä–∞–≥–º–µ–Ω—Ç –ø—Ä–µ–¥–æ—Å—Ç–∞–≤–ª—è–µ—Ç —Ö–æ—Ä–æ—à–∏–π –æ–±—â–µ–¥–æ—Å—Ç—É–ø–Ω—ã–π API. –≠—Ç–æ –æ–±–µ—Å–ø–µ—á–∏–≤–∞–µ—Ç—Å—è –ø—Ä–∞–≤–∏–ª–æ–º –ø—É–±–ª–∏—á–Ω–æ–≥–æ API –¥–ª—è —Å—Ä–µ–∑–æ–≤ (**public API rule on slices**):

> –ö–∞–∂–¥—ã–π —Å—Ä–µ–∑ (*slice*) (–∏ —Å–µ–≥–º–µ–Ω—Ç (*segment*) –Ω–∞ —Å–ª–æ—è—Ö, —É –∫–æ—Ç–æ—Ä—ã—Ö –Ω–µ—Ç —Ñ—Ä–∞–≥–º–µ–Ω—Ç–æ–≤) –¥–æ–ª–∂–µ–Ω —Å–æ–¥–µ—Ä–∂–∞—Ç—å –æ–ø—Ä–µ–¥–µ–ª–µ–Ω–∏–µ –æ–±—â–µ–¥–æ—Å—Ç—É–ø–Ω–æ–≥–æ API (*public API definition*).
> –ú–æ–¥—É–ª–∏ –∑–∞ –ø—Ä–µ–¥–µ–ª–∞–º–∏ —ç—Ç–æ–≥–æ —Å—Ä–µ–∑–∞ (*slice*) –∏–ª–∏ —Å–µ–≥–º–µ–Ω—Ç–∞ (*segment*) –º–æ–≥—É—Ç —Å—Å—ã–ª–∞—Ç—å—Å—è —Ç–æ–ª—å–∫–æ –Ω–∞ –æ–±—â–µ–¥–æ—Å—Ç—É–ø–Ω—ã–π API (*public API*), –Ω–æ –Ω–µ –Ω–∞ –≤–Ω—É—Ç—Ä–µ–Ω–Ω—é—é —Ñ–∞–π–ª–æ–≤—É—é —Å—Ç—Ä—É–∫—Ç—É—Ä—É (*slice*) –∏–ª–∏ —Å–µ–≥–º–µ–Ω—Ç–∞ (*segment*).

#### [Segments](https://feature-sliced.design/docs/reference/slices-segments#segments)

**Segments** (*—Å–µ–≥–º–µ–Ω—Ç—ã*) - —ç—Ç–æ —Ç—Ä–µ—Ç–∏–π –∏ –ø–æ—Å–ª–µ–¥–Ω–∏–π —É—Ä–æ–≤–µ–Ω—å –≤ –æ—Ä–≥–∞–Ω–∏–∑–∞—Ü–∏–æ–Ω–Ω–æ–π –∏–µ—Ä–∞—Ä—Ö–∏–∏, –∏ –∏—Ö –Ω–∞–∑–Ω–∞—á–µ–Ω–∏–µ - –≥—Ä—É–ø–ø–∏—Ä–æ–≤–∞—Ç—å –∫–æ–¥ –ø–æ –µ–≥–æ —Ç–µ—Ö–Ω–∏—á–µ—Å–∫–æ–π –ø—Ä–∏—Ä–æ–¥–µ.

–ï—Å—Ç—å –Ω–µ—Å–∫–æ–ª—å–∫–æ —Å—Ç–∞–Ω–¥–∞—Ä—Ç–∏–∑–∏—Ä–æ–≤–∞–Ω–Ω—ã—Ö –Ω–∞–∑–≤–∞–Ω–∏–π —Å–µ–≥–º–µ–Ω—Ç–æ–≤, —Å–º—ã—Å–ª –∫–æ—Ç–æ—Ä—ã—Ö –º–æ–∂–µ—Ç –º–µ–Ω—è—Ç—å—Å—è –æ—Ç –Ω–∞—Ö–æ–∂–¥–µ–Ω–∏—è –≤ –∫–∞–∂–¥–æ–º –∏–∑ —Å–ª–æ–µ–≤:

- `ui` ‚Äî –∫–æ–º–ø–æ–Ω–µ–Ω—Ç—ã UI, —Ñ—É–Ω–∫—Ü–∏–∏ —Ñ–æ—Ä–º–∞—Ç–∏—Ä–æ–≤–∞–Ω–∏—è –¥–∞–Ω–Ω—ã—Ö
<!-- –ù–µ —Å–æ–≤—Å–µ–º –ø–æ–Ω—è—Ç–Ω–æ –ø–æ—á–µ–º—É —Ñ—É–Ω–∫—Ü–∏–∏ —Ñ–æ—Ä–º–∞—Ç–∏—Ä–æ–≤–∞–Ω–∏—è –¥–∞–Ω–Ω—ã—Ö –¥–æ–ª–∂–Ω—ã –Ω–∞—Ö–æ–¥–∏—Ç—Å—è –≤ —Å–µ–≥–º–µ–Ω—Ç–µ `ui`  -->
- `model` ‚Äî –±–∏–∑–Ω–µ—Å‚Äî–ª–æ–≥–∏–∫–∞ –∏ —Ö—Ä–∞–Ω–µ–Ω–∏–µ –¥–∞–Ω–Ω—ã—Ö, –∞ —Ç–∞–∫–∂–µ —Ñ—É–Ω–∫—Ü–∏–∏ –¥–ª—è —É–ø—Ä–∞–≤–ª–µ–Ω–∏—è —ç—Ç–∏–º–∏ –¥–∞–Ω–Ω—ã–º–∏
- `lib` ‚Äî –≤—Å–ø–æ–º–æ–≥–∞—Ç–µ–ª—å–Ω—ã–π –∏ –∏–Ω—Ñ—Ä–∞—Å—Ç—Ä—É–∫—Ç—É—Ä–Ω—ã–π –∫–æ–¥
- `api` - –≤–∑–∞–∏–º–æ–¥–µ–π—Å—Ç–≤–∏–µ —Å –≤–Ω–µ—à–Ω–∏–º–∏ API, –≤–Ω—É—Ç—Ä–µ–Ω–Ω–∏–µ –º–µ—Ç–æ–¥—ã API

[–ü—Ä–∏–º–µ—Ä—ã –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏—è —Å—Ç–∞–Ω–¥–∞—Ä—Ç–∏–∑–∏—Ä–æ–≤–∞–Ω–Ω—ã—Ö —Å–µ–≥–º–µ–Ω—Ç–æ–≤ –≤ –∫–∞–∂–¥–æ–º —Å–ª–æ–µ](https://feature-sliced.design/docs/reference/slices-segments#examples)

–î—Ä—É–≥–∏–µ —Å–µ–≥–º–µ–Ω—Ç—ã (*custom segments*) —Ä–∞–∑—Ä–µ—à–∞—é—Ç—Å—è, –Ω–æ –∏—Ö —Å–ª–µ–¥—É–µ—Ç —Å–æ–∑–¥–∞–≤–∞—Ç—å —Å –æ—Å—Ç–æ—Ä–æ–∂–Ω–æ—Å—Ç—å—é. –ù–∞–∏–±–æ–ª–µ–µ —Ä–∞—Å–ø—Ä–æ—Å—Ç—Ä–∞–Ω–µ–Ω–Ω—ã–º–∏ –º–µ—Å—Ç–∞–º–∏ –¥–ª—è –¥—Ä—É–≥–∏—Ö —Å–µ–≥–º–µ–Ω—Ç—ã (*custom segments*) —è–≤–ª—è—é—Ç—Å—è —Å–ª–æ–π **App** –∏ —Å–ª–æ–π **Shared**, –≥–¥–µ —Ñ—Ä–∞–≥–º–µ–Ω—Ç—ã –Ω–µ –∏–º–µ—é—Ç —Å–º—ã—Å–ª–∞.

### [Public API](https://feature-sliced.design/docs/reference/public-api)

Public API (–æ–±—â–µ–¥–æ—Å—Ç—É–ø–Ω—ã–π API) –ø—Ä–µ–¥—Å—Ç–∞–≤–ª—è–µ—Ç —Å–æ–±–æ–π –µ–¥–∏–Ω—É—é —Ç–æ—á–∫—É –¥–æ—Å—Ç—É–ø–∞ –∫ –≤–æ–∑–º–æ–∂–Ω–æ—Å—Ç—è–º –º–æ–¥—É–ª—è –∏ –æ–ø—Ä–µ–¥–µ–ª—è—é—â–µ–≥–æ "–∫–æ–Ω—Ç—Ä–∞–∫—Ç" –≤–∑–∞–∏–º–æ–¥–µ–π—Å—Ç–≤–∏—è –º–æ–¥—É–ª—è —Å –≤–Ω–µ—à–Ω–∏–º –º–∏—Ä–æ–º.

–û–±—â–µ–¥–æ—Å—Ç—É–ø–Ω—ã–π API (*public API*) –¥–æ–ª–∂–µ–Ω –∫–æ–Ω—Ç—Ä–æ–ª–∏—Ä–æ–≤–∞—Ç—å –¥–æ—Å—Ç—É–ø (***control access***) –∫ —Å–æ–¥–µ—Ä–∂–∏–º–æ–º—É –º–æ–¥—É–ª—è

- –î—Ä—É–≥–∏–µ —á–∞—Å—Ç–∏ –ø—Ä–∏–ª–æ–∂–µ–Ω–∏—è –º–æ–≥—É—Ç –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å —Ç–æ–ª—å–∫–æ —Ç–µ –æ–±—ä–µ–∫—Ç—ã –º–æ–¥—É–ª—è, –∫–æ—Ç–æ—Ä—ã–µ –ø—Ä–µ–¥—Å—Ç–∞–≤–ª–µ–Ω—ã –≤ –æ–±—â–µ–¥–æ—Å—Ç—É–ø–Ω–æ–º –∏–Ω—Ç–µ—Ä—Ñ–µ–π—Å–µ (*public API*)
- –í–Ω—É—Ç—Ä–µ–Ω–Ω—è—è —á–∞—Å—Ç—å –º–æ–¥—É–ª—è –∑–∞ –ø—Ä–µ–¥–µ–ª–∞–º–∏ –æ–±—â–µ–¥–æ—Å—Ç—É–ø–Ω–æ–≥–æ –∏–Ω—Ç–µ—Ä—Ñ–µ–π—Å–∞ –¥–æ—Å—Ç—É–ø–Ω–∞ —Ç–æ–ª—å–∫–æ —Å–∞–º–æ–º—É –º–æ–¥—É–ª—é.

–û–±—â–µ–¥–æ—Å—Ç—É–ø–Ω—ã–π API (*public API*) –¥–æ–ª–∂–µ–Ω –±—ã—Ç—å —É—Å—Ç–æ–π—á–∏–≤—ã–º –∫ –∏–∑–º–µ–Ω–µ–Ω–∏—è–º (***sustainability for changes***) –≤–Ω—É—Ç—Ä–∏ –º–æ–¥—É–ª—è

- –ö—Ä–∏—Ç–∏—á–µ—Å–∫–∏–µ –∏–∑–º–µ–Ω–µ–Ω–∏—è –≤ –ø–æ–≤–µ–¥–µ–Ω–∏–∏ –º–æ–¥—É–ª—è –æ—Ç—Ä–∞–∂–∞—é—Ç—Å—è –≤ –∏–∑–º–µ–Ω–µ–Ω–∏–∏ –æ–±—â–µ–¥–æ—Å—Ç—É–ø–Ω–æ–≥–æ API (*public API*)

–ò–∑–º–µ–Ω–µ–Ω–∏–µ –≤–Ω—É—Ç—Ä–µ–Ω–Ω–µ–π —Å—Ç—Ä—É–∫—Ç—É—Ä—ã –Ω–µ –¥–æ–ª–∂–Ω–æ –ø—Ä–∏–≤–æ–¥–∏—Ç—å –∫ –∏–∑–º–µ–Ω–µ–Ω–∏—é –æ–±—â–µ–¥–æ—Å—Ç—É–ø–Ω–æ–≥–æ API (*public API*).

–û–±—â–µ–¥–æ—Å—Ç—É–ø–Ω—ã–π API (*public API*) –¥–æ–ª–∂–µ–Ω —Å–ø–æ—Å–æ–±—Å—Ç–≤–æ–≤–∞—Ç—å –ø—Ä–æ—Å—Ç–æ–π –∏ –≥–∏–±–∫–æ–π –∏–Ω—Ç–µ–≥—Ä–∞—Ü–∏–∏ (***easy and flexible integration***)

- –î–æ–ª–∂–µ–Ω –±—ã—Ç—å —É–¥–æ–±–µ–Ω –¥–ª—è –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏—è –æ—Å—Ç–∞–ª—å–Ω–æ–π —á–∞—Å—Ç—å—é –ø—Ä–∏–ª–æ–∂–µ–Ω–∏—è, –≤ —á–∞—Å—Ç–Ω–æ—Å—Ç–∏, –¥–ª—è —Ä–µ—à–µ–Ω–∏—è –ø—Ä–æ–±–ª–µ–º—ã –∫–æ–ª–ª–∏–∑–∏–π –∏–º–µ–Ω (*name collisions*)
