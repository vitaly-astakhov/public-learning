# State Management (Cookie)

–ò—Å–ø–æ–ª—å–∑—É—è –ø–æ–ª–µ –∑–∞–≥–æ–ª–æ–≤–∫–∞ `Set-Cookie`, HTTP-—Å–µ—Ä–≤–µ—Ä –º–æ–∂–µ—Ç –ø–µ—Ä–µ–¥–∞–≤–∞—Ç—å –ø–∞—Ä—ã –∏–º—è/–∑–Ω–∞—á–µ–Ω–∏–µ –∏ —Å–≤—è–∑–∞–Ω–Ω—ã–µ –º–µ—Ç–∞–¥–∞–Ω–Ω—ã–µ (–Ω–∞–∑—ã–≤–∞–µ–º—ã–µ —Ñ–∞–π–ª–∞–º–∏ cookie) –ø–æ–ª—å–∑–æ–≤–∞—Ç–µ–ª—å—Å–∫–æ–º—É –∞–≥–µ–Ω—Ç—É. –ö–æ–≥–¥–∞ –ø–æ–ª—å–∑–æ–≤–∞—Ç–µ–ª—å—Å–∫–∏–π –∞–≥–µ–Ω—Ç –≤—ã–ø–æ–ª–Ω—è–µ—Ç –ø–æ—Å–ª–µ–¥—É—é—â–∏–µ –∑–∞–ø—Ä–æ—Å—ã –∫ —Å–µ—Ä–≤–µ—Ä—É, –ø–æ–ª—å–∑–æ–≤–∞—Ç–µ–ª—å—Å–∫–∏–π –∞–≥–µ–Ω—Ç –∏—Å–ø–æ–ª—å–∑—É–µ—Ç –º–µ—Ç–∞–¥–∞–Ω–Ω—ã–µ –∏ –¥—Ä—É–≥—É—é –∏–Ω—Ñ–æ—Ä–º–∞—Ü–∏—é, —á—Ç–æ–±—ã –æ–ø—Ä–µ–¥–µ–ª–∏—Ç—å, —Å–ª–µ–¥—É–µ—Ç –ª–∏ –≤–æ–∑–≤—Ä–∞—â–∞—Ç—å –ø–∞—Ä—ã –∏–º—è/–∑–Ω–∞—á–µ–Ω–∏–µ –≤ –ø–æ–ª–µ –∑–∞–≥–æ–ª–æ–≤–∫–∞ `Cookie`.


–î–ª—è —Ö—Ä–∞–Ω–µ–Ω–∏—è —Å–æ—Å—Ç–æ—è–Ω–∏—è –∏—Å—Ö–æ–¥–Ω—ã–π —Å–µ—Ä–≤–µ—Ä –≤–∫–ª—é—á–∞–µ—Ç –ø–æ–ª–µ –∑–∞–≥–æ–ª–æ–≤–∫–∞ `Set-Cookie` –≤ –æ—Ç–≤–µ—Ç HTTP. –í –ø–æ—Å–ª–µ–¥—É—é—â–∏—Ö –∑–∞–ø—Ä–æ—Å–∞—Ö –ø–æ–ª—å–∑–æ–≤–∞—Ç–µ–ª—å—Å–∫–∏–π –∞–≥–µ–Ω—Ç –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç –ø–æ–ª–µ –∑–∞–≥–æ–ª–æ–≤–∫–∞ –∑–∞–ø—Ä–æ—Å–∞ `Cookie` –∏—Å—Ö–æ–¥–Ω–æ–º—É —Å–µ—Ä–≤–µ—Ä—É. –ü–æ–ª–µ –∑–∞–≥–æ–ª–æ–≤–∫–∞ `Cookie` —Å–æ–¥–µ—Ä–∂–∏—Ç —Ñ–∞–π–ª—ã cookie, –∫–æ—Ç–æ—Ä—ã–µ –ø–æ–ª—å–∑–æ–≤–∞—Ç–µ–ª—å—Å–∫–∏–π –∞–≥–µ–Ω—Ç –ø–æ–ª—É—á–∏–ª –≤ –ø—Ä–µ–¥—ã–¥—É—â–∏—Ö –ø–æ–ª—è—Ö –∑–∞–≥–æ–ª–æ–≤–∫–∞ `Set-Cookie`. –ò—Å—Ö–æ–¥–Ω—ã–π —Å–µ—Ä–≤–µ—Ä (**origin server**) –º–æ–∂–µ—Ç –∏–≥–Ω–æ—Ä–∏—Ä–æ–≤–∞—Ç—å –ø–æ–ª–µ –∑–∞–≥–æ–ª–æ–≤–∫–∞ `Cookie` –∏–ª–∏ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å –µ–≥–æ —Å–æ–¥–µ—Ä–∂–∏–º–æ–µ –¥–ª—è —Ü–µ–ª–µ–π, –æ–ø—Ä–µ–¥–µ–ª—è–µ–º—ã—Ö –ø—Ä–∏–ª–æ–∂–µ–Ω–∏–µ–º.

–ö–æ–≥–¥–∞ –ø–æ–ª—å–∑–æ–≤–∞—Ç–µ–ª—å—Å–∫–∏–π –∞–≥–µ–Ω—Ç –ø–æ–ª—É—á–∞–µ—Ç –ø–æ–ª–µ –∑–∞–≥–æ–ª–æ–≤–∫–∞ `Set-Cookie`, –ø–æ–ª—å–∑–æ–≤–∞—Ç–µ–ª—å—Å–∫–∏–π –∞–≥–µ–Ω—Ç —Å–æ—Ö—Ä–∞–Ω—è–µ—Ç —Ñ–∞–π–ª cookie –≤–º–µ—Å—Ç–µ —Å –µ–≥–æ –∞—Ç—Ä–∏–±—É—Ç–∞–º–∏. –í–ø–æ—Å–ª–µ–¥—Å—Ç–≤–∏–∏, –∫–æ–≥–¥–∞ –ø–æ–ª—å–∑–æ–≤–∞—Ç–µ–ª—å—Å–∫–∏–π –∞–≥–µ–Ω—Ç –≤—ã–ø–æ–ª–Ω—è–µ—Ç HTTP-–∑–∞–ø—Ä–æ—Å, –ø–æ–ª—å–∑–æ–≤–∞—Ç–µ–ª—å—Å–∫–∏–π –∞–≥–µ–Ω—Ç –≤–∫–ª—é—á–∞–µ—Ç –ø—Ä–∏–º–µ–Ω–∏–º—ã–µ —Ñ–∞–π–ª—ã cookie —Å –Ω–µ–∏—Å—Ç–µ–∫—à–∏–º —Å—Ä–æ–∫–æ–º –¥–µ–π—Å—Ç–≤–∏—è –≤ –ø–æ–ª–µ –∑–∞–≥–æ–ª–æ–≤–∫–∞ `Cookie`.

### Server

–ò—Å—Ö–æ–¥–Ω—ã–µ —Å–µ—Ä–≤–µ—Ä—ã **       (MAY)** –æ—Ç–ø—Ä–∞–≤–ª—è—Ç—å –ø–æ–ª–µ –∑–∞–≥–æ–ª–æ–≤–∫–∞ –æ—Ç–≤–µ—Ç–∞ `Set-Cookie` —Å –ª—é–±—ã–º –æ—Ç–≤–µ—Ç–æ–º. –ò—Å—Ö–æ–¥–Ω—ã–π —Å–µ—Ä–≤–µ—Ä –º–æ–∂–µ—Ç –≤–∫–ª—é—á–∞—Ç—å –Ω–µ—Å–∫–æ–ª—å–∫–æ –ø–æ–ª–µ–π –∑–∞–≥–æ–ª–æ–≤–∫–∞ `Set-Cookie` –≤ –æ–¥–∏–Ω –æ—Ç–≤–µ—Ç. –ù–∞–ª–∏—á–∏–µ –ø–æ–ª—è –∑–∞–≥–æ–ª–æ–≤–∫–∞ `Cookie` –∏–ª–∏ `Set-Cookie` –Ω–µ –ø—Ä–µ–ø—è—Ç—Å—Ç–≤—É–µ—Ç —Å–æ—Ö—Ä–∞–Ω–µ–Ω–∏—é –∏ –ø–æ–≤—Ç–æ—Ä–Ω–æ–º—É –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏—é –æ—Ç–≤–µ—Ç–∞ HTTP-–∫—ç—à–∞–º–∏.

–ò—Å—Ö–æ–¥–Ω—ã–µ —Å–µ—Ä–≤–µ—Ä—ã **–ù–ï –î–û–õ–ñ–ù–´ (SHOULD NOT)** –æ–±—ä–µ–¥–∏–Ω—è—Ç—å –Ω–µ—Å–∫–æ–ª—å–∫–æ –ø–æ–ª–µ–π –∑–∞–≥–æ–ª–æ–≤–∫–∞ `Set-Cookie` –≤ –æ–¥–Ω–æ –ø–æ–ª–µ –∑–∞–≥–æ–ª–æ–≤–∫–∞. –≠—Ç–æ —è–≤–ª—è–µ—Ç—Å—è [–∏—Å–∫–ª—é—á–µ–Ω–∏–µ–º –∏–∑ –ø—Ä–∞–≤–∏–ª](https://datatracker.ietf.org/doc/html/rfc9110#name-field-order).

### User Agent

–ü–æ–ª—å–∑–æ–≤–∞—Ç–µ–ª—å—Å–∫–∏–µ –∞–≥–µ–Ω—Ç—ã **–ú–û–ì–£–¢ (MAY)** –∏–≥–Ω–æ—Ä–∏—Ä–æ–≤–∞—Ç—å –ø–æ–ª—è –∑–∞–≥–æ–ª–æ–≤–∫–∞ `Set-Cookie` –Ω–∞ –æ—Å–Ω–æ–≤–µ –∫–æ–¥–æ–≤ —Å–æ—Å—Ç–æ—è–Ω–∏—è –æ—Ç–≤–µ—Ç–∞ –∏–ª–∏ –ø–æ–ª–∏—Ç–∏–∫–∏ —Ñ–∞–π–ª–æ–≤ cookie (**cookie policy**) –ø–æ–ª—å–∑–æ–≤–∞—Ç–µ–ª—å—Å–∫–æ–≥–æ –∞–≥–µ–Ω—Ç–∞.

–ï—Å–ª–∏ –ø–æ–ª—å–∑–æ–≤–∞—Ç–µ–ª—å—Å–∫–∏–π –∞–≥–µ–Ω—Ç –ø–æ–ª—É—á–∞–µ—Ç –Ω–æ–≤—ã–π —Ñ–∞–π–ª cookie —Å —Ç–µ–º –∂–µ –∏–º–µ–Ω–µ–º —Ñ–∞–π–ª–∞ cookie (cookie-name), –∑–Ω–∞—á–µ–Ω–∏–µ–º –¥–æ–º–µ–Ω–∞ (domain-value) –∏ –∑–Ω–∞—á–µ–Ω–∏–µ–º –ø—É—Ç–∏ (path-value), —á—Ç–æ –∏ —Ñ–∞–π–ª cookie, –∫–æ—Ç–æ—Ä—ã–π –æ–Ω —É–∂–µ —Å–æ—Ö—Ä–∞–Ω–∏–ª, —Å—É—â–µ—Å—Ç–≤—É—é—â–∏–π —Ñ–∞–π–ª cookie —É–¥–∞–ª—è–µ—Ç—Å—è –∏ –∑–∞–º–µ–Ω—è–µ—Ç—Å—è –Ω–æ–≤—ã–º —Ñ–∞–π–ª–æ–º cookie. –û–±—Ä–∞—Ç–∏—Ç–µ –≤–Ω–∏–º–∞–Ω–∏–µ, —á—Ç–æ —Å–µ—Ä–≤–µ—Ä—ã –º–æ–≥—É—Ç —É–¥–∞–ª—è—Ç—å —Ñ–∞–π–ª—ã cookie, –æ—Ç–ø—Ä–∞–≤–ª—è—è –ø–æ–ª—å–∑–æ–≤–∞—Ç–µ–ª—å—Å–∫–æ–º—É –∞–≥–µ–Ω—Ç—É –Ω–æ–≤—ã–π —Ñ–∞–π–ª cookie —Å –∞—Ç—Ä–∏–±—É—Ç–æ–º Expires —Å–æ –∑–Ω–∞—á–µ–Ω–∏–µ–º –≤ –ø—Ä–æ—à–ª–æ–º.

–ü–æ–ª—å–∑–æ–≤–∞—Ç–µ–ª—å—Å–∫–∏–µ –∞–≥–µ–Ω—Ç—ã **–î–û–õ–ñ–ù–´ (SHOULD)** –ø—Ä–µ–¥–æ—Å—Ç–∞–≤–ª—è—Ç—å –ø–æ–ª—å–∑–æ–≤–∞—Ç–µ–ª—é –º–µ—Ö–∞–Ω–∏–∑–º –¥–ª—è –≤—ã–∫–ª—é—á–µ–Ω–∏—è (disabling) —Ñ–∞–π–ª–æ–≤ cookie. –ï—Å–ª–∏ –ø–æ–ª—å–∑–æ–≤–∞—Ç–µ–ª—å —Ä–µ—à–∏–ª "–≤—ã–∫–ª—é—á–∏—Ç—å" –∫—É–∫–∏, —Ç–æ –ø–æ–ª—å–∑–æ–≤–∞—Ç–µ–ª—å—Å–∫–∏–π –∞–≥–µ–Ω—Ç **–ù–ï –î–û–õ–ñ–ï–ù (MUST NOT)** –æ–±—Ä–∞–±–∞—Ç—ã–≤–∞—Ç—å —Ñ–∞–π–ª—ã cookie —á–µ—Ä–µ–∑ –ø–æ–ª–µ `Set-Cookie`, –∏ **–ù–ï –î–û–õ–ñ–ï–ù (MUST NOT)** –æ—Ç—Å—ã–ª–∞—Ç—å —Ñ–∞–π–ª—ã cookie —á–µ—Ä–µ–∑ –ø–æ–ª–µ `Cookie`

–ü–æ–ª—å–∑–æ–≤–∞—Ç–µ–ª—å—Å–∫–∏–π –∞–≥–µ–Ω—Ç **–î–û–õ–ñ–ï–ù (MUST )** –æ–≥—Ä–∞–Ω–∏—á–∏—Ç—å –º–∞–∫—Å–∏–º–∞–ª—å–Ω–æ–µ –∑–Ω–∞—á–µ–Ω–∏–µ –∞—Ç—Ä–∏–±—É—Ç–∞ `Expires`. –û–≥—Ä–∞–Ω–∏—á–µ–Ω–∏–µ **–ù–ï –î–û–õ–ñ–ù–û (SHOULD NOT)** –ø—Ä–µ–≤—ã—à–∞—Ç—å 400 –¥–Ω–µ–π (34560000 —Å–µ–∫—É–Ω–¥) –≤ –±—É–¥—É—â–µ–º. **–†–ï–ö–û–ú–ï–ù–î–£–ï–ú–´–ô (RECOMMENDED)** –ª–∏–º–∏—Ç —Å–æ—Å—Ç–∞–≤–ª—è–µ—Ç 400 –¥–Ω–µ–π –≤ –±—É–¥—É—â–µ–º, –Ω–æ [–ø–æ–ª—å–∑–æ–≤–∞—Ç–µ–ª—å—Å–∫–∏–π –∞–≥–µ–Ω—Ç **–ú–û–ñ–ï–¢ (MAY)** —Å–∫–æ—Ä—Ä–µ–∫—Ç–∏—Ä–æ–≤–∞—Ç—å —ç—Ç–æ—Ç –ª–∏–º–∏—Ç](https://www.ietf.org/archive/id/draft-ietf-httpbis-rfc6265bis-13.html#cookie-policy). –ê—Ç—Ä–∏–±—É—Ç—ã `Expires`, –∫–æ—Ç–æ—Ä—ã–µ –ø—Ä–µ–≤—ã—à–∞—é—Ç –ª–∏–º–∏—Ç, **–î–û–õ–ñ–ù–´ (MUST)** –±—ã—Ç—å —É–º–µ–Ω—å—à–µ–Ω—ã –¥–æ —ç—Ç–æ–≥–æ –ø—Ä–µ–¥–µ–ª–∞.

### Max-Age
**Max-Age** - —ç—Ç–æ—Ç cookie –∞—Ç—Ç—Ä–∏–±—É—Ç —É–∫–∞–∑—ã–≤–∞–µ—Ç –º–∞–∫—Å–∏–º–∞–ª—å–Ω–æ–µ –≤—Ä–µ–º—è –∂–∏–∑–Ω–∏ —Ñ–∞–π–ª–∞ cookie, –ø—Ä–µ–¥—Å—Ç–∞–≤–ª–µ–Ω–Ω–æ–µ –≤ –≤–∏–¥–µ –∫–æ–ª–∏—á–µ—Å—Ç–≤–∞ —Å–µ–∫—É–Ω–¥ –¥–æ –∏—Å—Ç–µ—á–µ–Ω–∏—è —Å—Ä–æ–∫–∞ –¥–µ–π—Å—Ç–≤–∏—è —Ñ–∞–π–ª–∞ cookie

–ü–æ–ª—å–∑–æ–≤–∞—Ç–µ–ª—å—Å–∫–∏–π –∞–≥–µ–Ω—Ç **–î–û–õ–ñ–ï–ù (MUST )** –æ–≥—Ä–∞–Ω–∏—á–∏—Ç—å –º–∞–∫—Å–∏–º–∞–ª—å–Ω–æ–µ –∑–Ω–∞—á–µ–Ω–∏–µ –∞—Ç—Ä–∏–±—É—Ç–∞ `Max-Age`. –û–≥—Ä–∞–Ω–∏—á–µ–Ω–∏–µ **–ù–ï –î–û–õ–ñ–ù–û (SHOULD NOT)** –ø—Ä–µ–≤—ã—à–∞—Ç—å 400 –¥–Ω–µ–π (34560000 —Å–µ–∫—É–Ω–¥) –≤ –±—É–¥—É—â–µ–º. **–†–ï–ö–û–ú–ï–ù–î–£–ï–ú–´–ô (RECOMMENDED)** –ª–∏–º–∏—Ç —Å–æ—Å—Ç–∞–≤–ª—è–µ—Ç 400 –¥–Ω–µ–π –≤ –±—É–¥—É—â–µ–º, –Ω–æ [–ø–æ–ª—å–∑–æ–≤–∞—Ç–µ–ª—å—Å–∫–∏–π –∞–≥–µ–Ω—Ç **–ú–û–ñ–ï–¢ (MAY)** —Å–∫–æ—Ä—Ä–µ–∫—Ç–∏—Ä–æ–≤–∞—Ç—å —ç—Ç–æ—Ç –ª–∏–º–∏—Ç](https://www.ietf.org/archive/id/draft-ietf-httpbis-rfc6265bis-13.html#cookie-policy). –ê—Ç—Ä–∏–±—É—Ç—ã `Max-Age`, –∫–æ—Ç–æ—Ä—ã–µ –ø—Ä–µ–≤—ã—à–∞—é—Ç –ª–∏–º–∏—Ç, **–î–û–õ–ñ–ù–´ (MUST)** –±—ã—Ç—å —É–º–µ–Ω—å—à–µ–Ω—ã –¥–æ —ç—Ç–æ–≥–æ –ø—Ä–µ–¥–µ–ª–∞.

–ù–µ–∫–æ—Ç–æ—Ä—ã–µ —Å—É—â–µ—Å—Ç–≤—É—é—â–∏–µ –ø–æ–ª—å–∑–æ–≤–∞—Ç–µ–ª—å—Å–∫–∏–µ –∞–≥–µ–Ω—Ç—ã –Ω–µ –ø–æ–¥–¥–µ—Ä–∂–∏–≤–∞—é—Ç –∞—Ç—Ä–∏–±—É—Ç `Max-Age`. –ü–æ–ª—å–∑–æ–≤–∞—Ç–µ–ª—å—Å–∫–∏–µ –∞–≥–µ–Ω—Ç—ã, –∫–æ—Ç–æ—Ä—ã–µ –Ω–µ –ø–æ–¥–¥–µ—Ä–∂–∏–≤–∞—é—Ç –∞—Ç—Ä–∏–±—É—Ç `Max-Age`, –∏–≥–Ω–æ—Ä–∏—Ä—É—é—Ç —ç—Ç–æ—Ç –∞—Ç—Ä–∏–±—É—Ç.

–ï—Å–ª–∏ —Ñ–∞–π–ª cookie –∏–º–µ–µ—Ç –∫–∞–∫ –∞—Ç—Ä–∏–±—É—Ç `Max-Age`, —Ç–∞–∫ –∏ –∞—Ç—Ä–∏–±—É—Ç `Expires`, –∞—Ç—Ä–∏–±—É—Ç `Max-Age` –∏–º–µ–µ—Ç –ø—Ä–∏–æ—Ä–∏—Ç–µ—Ç –∏ –∫–æ–Ω—Ç—Ä–æ–ª–∏—Ä—É–µ—Ç –¥–∞—Ç—É –∏—Å—Ç–µ—á–µ–Ω–∏—è —Å—Ä–æ–∫–∞ –¥–µ–π—Å—Ç–≤–∏—è —Ñ–∞–π–ª–∞ cookie. –ï—Å–ª–∏ —Ñ–∞–π–ª cookie –Ω–µ –∏–º–µ–µ—Ç –Ω–∏ –∞—Ç—Ä–∏–±—É—Ç–∞ `Max-Age`, –Ω–∏ –∞—Ç—Ä–∏–±—É—Ç–∞ `Expires`, –ø–æ–ª—å–∑–æ–≤–∞—Ç–µ–ª—å—Å–∫–∏–π –∞–≥–µ–Ω—Ç –±—É–¥–µ—Ç —Ö—Ä–∞–Ω–∏—Ç—å —Ñ–∞–π–ª cookie –¥–æ —Ç–µ—Ö –ø–æ—Ä, –ø–æ–∫–∞ ¬´—Ç–µ–∫—É—â–∏–π —Å–µ–∞–Ω—Å –Ω–µ –∑–∞–≤–µ—Ä—à–∏—Ç—Å—è¬ª (–∫–∞–∫ –æ–ø—Ä–µ–¥–µ–ª–µ–Ω–æ –ø–æ–ª—å–∑–æ–≤–∞—Ç–µ–ª—å—Å–∫–∏–º –∞–≥–µ–Ω—Ç–æ–º).
