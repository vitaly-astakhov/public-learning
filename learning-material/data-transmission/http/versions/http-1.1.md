–ö–ª–∏–µ–Ω—Ç, –∫–æ—Ç–æ—Ä—ã–π –ø–æ–ª—É—á–∞–µ—Ç –Ω–µ–ø–æ–ª–Ω–æ–µ –æ—Ç–≤–µ—Ç–Ω–æ–µ —Å–æ–æ–±—â–µ–Ω–∏–µ, —á—Ç–æ –º–æ–∂–µ—Ç –ø—Ä–æ–∏–∑–æ–π—Ç–∏ –ø—Ä–∏ –ø—Ä–µ–∂–¥–µ–≤—Ä–µ–º–µ–Ω–Ω–æ–º –∑–∞–∫—Ä—ã—Ç–∏–∏ —Å–æ–µ–¥–∏–Ω–µ–Ω–∏—è –∏–ª–∏ –ø—Ä–∏ —Å–±–æ–µ –¥–µ–∫–æ–¥–∏—Ä–æ–≤–∞–Ω–∏—è –ø—Ä–µ–¥–ø–æ–ª–æ–∂–∏—Ç–µ–ª—å–Ω–æ —Ñ—Ä–∞–≥–º–µ–Ω—Ç–∏—Ä–æ–≤–∞–Ω–Ω–æ–≥–æ –∫–æ–¥–∏—Ä–æ–≤–∞–Ω–∏—è –ø–µ—Ä–µ–¥–∞—á–∏, **–î–û–õ–ñ–ï–ù** –∑–∞–ø–∏—Å–∞—Ç—å —Å–æ–æ–±—â–µ–Ω–∏–µ –∫–∞–∫ –Ω–µ–ø–æ–ª–Ω–æ–µ.

–¢–µ–ª–æ —Å–æ–æ–±—â–µ–Ω–∏—è, –≤ –∫–æ—Ç–æ—Ä–æ–º –∏—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è –∫–æ–¥–∏—Ä–æ–≤–∞–Ω–∏–µ –ø–µ—Ä–µ–¥–∞—á–∏ –ø–æ —á–∞—Å—Ç—è–º (**chunked transfer coding**), —è–≤–ª—è–µ—Ç—Å—è –Ω–µ–ø–æ–ª–Ω—ã–º, –µ—Å–ª–∏ –Ω–µ –ø–æ–ª—É—á–µ–Ω —Ñ—Ä–∞–≥–º–µ–Ω—Ç –Ω—É–ª–µ–≤–æ–≥–æ —Ä–∞–∑–º–µ—Ä–∞, –∑–∞–≤–µ—Ä—à–∞—é—â–∏–π –∫–æ–¥–∏—Ä–æ–≤–∞–Ω–∏–µ.

–°–æ–æ–±—â–µ–Ω–∏–µ, –≤ –∫–æ—Ç–æ—Ä–æ–º –∏—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è –¥–æ–ø—É—Å—Ç–∏–º—ã–π `Content-Length`, —è–≤–ª—è–µ—Ç—Å—è –Ω–µ–ø–æ–ª–Ω—ã–º, –µ—Å–ª–∏ —Ä–∞–∑–º–µ—Ä –ø–æ–ª—É—á–µ–Ω–Ω–æ–≥–æ —Ç–µ–ª–∞ —Å–æ–æ–±—â–µ–Ω–∏—è (–≤ –æ–∫—Ç–µ—Ç–∞—Ö) –º–µ–Ω—å—à–µ –∑–Ω–∞—á–µ–Ω–∏—è, –∑–∞–¥–∞–Ω–Ω–æ–≥–æ `Content-Length`.

### [Connection Management](https://www.rfc-editor.org/rfc/rfc9112.html#name-connection-management)

–û–±–º–µ–Ω —Å–æ–æ–±—â–µ–Ω–∏—è–º–∏ HTTP –Ω–µ –∑–∞–≤–∏—Å–∏—Ç –æ—Ç –±–∞–∑–æ–≤–æ–≥–æ –ø—Ä–æ—Ç–æ–∫–æ–ª–∞ —Å–æ–µ–¥–∏–Ω–µ–Ω–∏—è —Ç—Ä–∞–Ω—Å–ø–æ—Ä—Ç–Ω–æ–≥–æ –∏–ª–∏ —Å–µ–∞–Ω—Å–æ–≤–æ–≥–æ —É—Ä–æ–≤–Ω—è. HTTP –ø—Ä–µ–¥–ø–æ–ª–∞–≥–∞–µ—Ç —Ç–æ–ª—å–∫–æ –Ω–∞–¥–µ–∂–Ω—ã–π —Ç—Ä–∞–Ω—Å–ø–æ—Ä—Ç —Å —É–ø–æ—Ä—è–¥–æ—á–µ–Ω–Ω–æ–π –¥–æ—Å—Ç–∞–≤–∫–æ–π –∑–∞–ø—Ä–æ—Å–æ–≤ –∏ —Å–æ–æ—Ç–≤–µ—Ç—Å—Ç–≤—É—é—â–µ–π —É–ø–æ—Ä—è–¥–æ—á–µ–Ω–Ω–æ–π –¥–æ—Å—Ç–∞–≤–∫–æ–π –æ—Ç–≤–µ—Ç–æ–≤.

HTTP/1.1 –ø–æ —É–º–æ–ª—á–∞–Ω–∏—é –∏—Å–ø–æ–ª—å–∑—É–µ—Ç ¬´–ø–æ—Å—Ç–æ—è–Ω–Ω—ã–µ —Å–æ–µ–¥–∏–Ω–µ–Ω–∏—è¬ª (**persistent connections**), –ø–æ–∑–≤–æ–ª—è—è –ø–µ—Ä–µ–¥–∞–≤–∞—Ç—å –Ω–µ—Å–∫–æ–ª—å–∫–æ –∑–∞–ø—Ä–æ—Å–æ–≤ –∏ –æ—Ç–≤–µ—Ç–æ–≤ –ø–æ –æ–¥–Ω–æ–º—É —Å–æ–µ–¥–∏–Ω–µ–Ω–∏—é. –†–µ–∞–ª–∏–∑–∞—Ü–∏–∏ HTTP **–î–û–õ–ñ–ù–´ (SHOULD)** –ø–æ–¥–¥–µ—Ä–∂–∏–≤–∞—Ç—å –ø–æ—Å—Ç–æ—è–Ω–Ω—ã–µ —Å–æ–µ–¥–∏–Ω–µ–Ω–∏—è.

–ü–æ–ª—É—á–∞—Ç–µ–ª—å –æ–ø—Ä–µ–¥–µ–ª—è–µ—Ç, —è–≤–ª—è–µ—Ç—Å—è –ª–∏ —Å–æ–µ–¥–∏–Ω–µ–Ω–∏–µ –ø–æ—Å—Ç–æ—è–Ω–Ω—ã–º (**persistent connections**) –∏–ª–∏ –Ω–µ—Ç, –Ω–∞ –æ—Å–Ω–æ–≤–µ –≤–µ—Ä—Å–∏–∏ –ø—Ä–æ—Ç–æ–∫–æ–ª–∞ –∏ –ø–æ–ª—è –∑–∞–≥–æ–ª–æ–≤–∫–∞ [`Connection`](https://www.rfc-editor.org/rfc/rfc9110#section-7.6.1) –≤ –ø–æ—Å–ª–µ–¥–Ω–µ–º –ø–æ–ª—É—á–µ–Ω–Ω–æ–º —Å–æ–æ–±—â–µ–Ω–∏–∏, –µ—Å–ª–∏ —Ç–∞–∫–æ–≤–æ–µ –∏–º–µ–µ—Ç—Å—è.


–ü–æ–ª—É—á–∞—Ç–µ–ª—å –æ–ø—Ä–µ–¥–µ–ª—è–µ—Ç, —è–≤–ª—è–µ—Ç—Å—è –ª–∏ —Å–æ–µ–¥–∏–Ω–µ–Ω–∏–µ –ø–æ—Å—Ç–æ—è–Ω–Ω—ã–º –∏–ª–∏ –Ω–µ—Ç, –Ω–∞ –æ—Å–Ω–æ–≤–µ –≤–µ—Ä—Å–∏–∏ –ø—Ä–æ—Ç–æ–∫–æ–ª–∞ –∏ –ø–æ–ª—è –∑–∞–≥–æ–ª–æ–≤–∫–∞ [`Connection`](https://www.rfc-editor.org/rfc/rfc9110#section-7.6.1) –≤ –ø–æ—Å–ª–µ–¥–Ω–µ–º –ø–æ–ª—É—á–µ–Ω–Ω–æ–º —Å–æ–æ–±—â–µ–Ω–∏–∏, –µ—Å–ª–∏ —Ç–∞–∫–æ–≤–æ–µ –∏–º–µ–µ—Ç—Å—è:

- –ï—Å–ª–∏ –ø–æ–ª—É—á–µ–Ω–Ω—ã–π –ø—Ä–æ—Ç–æ–∫–æ–ª ‚Äî HTTP/1.1 (–∏–ª–∏ –±–æ–ª–µ–µ –ø–æ–∑–¥–Ω–∏–π), —Å–æ–µ–¥–∏–Ω–µ–Ω–∏–µ —Å–æ—Ö—Ä–∞–Ω–∏—Ç—Å—è –ø–æ—Å–ª–µ —Ç–µ–∫—É—â–µ–≥–æ –æ—Ç–≤–µ—Ç–∞.
- –ï—Å–ª–∏ –ø–æ–ª—É—á–µ–Ω–Ω—ã–º –ø—Ä–æ—Ç–æ–∫–æ–ª–æ–º —è–≤–ª—è–µ—Ç—Å—è HTTP/1.0, –ø—Ä–∏—Å—É—Ç—Å—Ç–≤—É–µ—Ç –æ–ø—Ü–∏—è –ø–æ–¥–∫–ª—é—á–µ–Ω–∏—è "keep-alive", –ª–∏–±–æ –ø–æ–ª—É—á–∞—Ç–µ–ª—å –Ω–µ —è–≤–ª—è–µ—Ç—Å—è –ø—Ä–æ–∫—Å–∏-—Å–µ—Ä–≤–µ—Ä–æ–º, –ª–∏–±–æ —Å–æ–æ–±—â–µ–Ω–∏–µ —è–≤–ª—è–µ—Ç—Å—è –æ—Ç–≤–µ—Ç–æ–º, –∏ –ø–æ–ª—É—á–∞—Ç–µ–ª—å –∂–µ–ª–∞–µ—Ç —Å–æ–±–ª—é–¥–∞—Ç—å –º–µ—Ö–∞–Ω–∏–∑–º HTTP/1.0 "keep-alive", —Å–æ–µ–¥–∏–Ω–µ–Ω–∏–µ —Å–æ—Ö—Ä–∞–Ω–∏—Ç—Å—è –ø–æ—Å–ª–µ —Ç–µ–∫—É—â–µ–≥–æ –æ—Ç–≤–µ—Ç–∞
